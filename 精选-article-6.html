DeepSeek API 推出夜间错峰优惠，R1 价格降至 1/4

📝 详细摘要

DeepSeek 开放平台为了提升用户体验和优化资源利用率，分摊服务器压力，推出了 API 错峰优惠活动。该活动在北京时间每日 00:30 至 08:30 的夜间时段生效，在此期间，DeepSeek-V3 的 API 调用价格降至原价的 50%，DeepSeek-R1 更是降至 25%。DeepSeek 鼓励用户充分利用夜间空闲时段，以更低的成本享受更流畅的服务，从而实现开发者与平台的双赢。

💡 主要观点

  1. DeepSeek API 夜间错峰优惠，R1 价格降至 1/4，降低开发者成本 通过在服务器空闲时段降低 API 调用价格，特别是将 R1 价格降至四分之一，有效减少了开发者的经济负担，鼓励更多用户体验 DeepSeek 的服务。
  2. DeepSeek 鼓励用户利用空闲时段提升使用体验 在夜间时段，服务器负载较低，用户可以享受到更流畅、更快速的 API 响应，优化整体使用体验。
