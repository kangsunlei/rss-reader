ModelScope 魔搭 25 年 2 月版本发布月报

2/17/2025, 1:17:39 PM

📌 一句话摘要

ModelScope 魔搭社区 2 月版本发布月报，重点更新包括 AIGC 专区功能迭代、Langflow 创空间上线，以及针对 Reasoning 模型的开源 SDK 与工具链增强。

📝 详细摘要

ModelScope 魔搭社区发布了 2 月版本月报，涵盖了站点更新和开源 SDK 与工具链更新两大部分。站点更新主要集中在提升用户体验和 AIGC 能力上，包括模型血缘树系谱的发布，更直观地展示模型关系；API-Inference 与 ModelScope 站点的集成，方便开发者获取模型及示例代码；AIGC 模型页面改版，更贴近用户习惯；AIGC 专业生图功能迭代，提升创作体验和效率，例如新增差异种子功能、支持更高分辨率图像输出、提示词反推功能以及 ADetailer 增强等；以及全新上线 Langflow 创空间，简化 Agent 和 RAG 应用创建。开源 SDK 与工具链更新则主要围绕 Reasoning 模型展开，包括 SWIFT 对 GRPO 训练的支持，EvalScope 支持 R1 相关数据集评测，ModelScope SDK 和 SWIFT 版本更新，以及 EvalScope 评测报告可视化等。整体来看，本次更新旨在提升 ModelScope 平台易用性，并加强对新兴 Reasoning 模型和 AIGC 领域的支持，特别是对 DeepSeek-R1 系列模型的支持。

💡 主要观点

  1. 模型血缘树系谱使模型发现更直观，方便溯源 通过图形化展示模型之间的关联，帮助开发者更好地理解和利用 ModelScope 上的模型资源。
  2. API-Inference 集成方便开发者定位模型并获取示例 将 API-Inference 功能与 ModelScope 站点集成，降低了开发者使用 API 的门槛，提升了开发效率。
  3. AIGC 专业生图功能进行了诸多迭代 通过新增差异种子、支持更高分辨率、提示词反推、ADetailer 增强等功能，显著提升了用户的创作体验和效率。
  4. Langflow 创空间简化 Agent 和 RAG 应用创建 通过拖拉拽操作，降低了 Agent 和 RAG 应用的开发难度，让更多开发者能够参与到 AI 应用的创新中。
  5. 开源工具链针对 Reasoning 模型提供配套能力 SWIFT、EvalScope 等工具链的更新，为 Reasoning 模型的研究和应用提供了更全面的支持，推动了该领域的发展。

💬 文章金句

  • 让模型的发现更加直观，方便开发者从单模型入口进行拓展，实现更好的模型树溯源。
  • 支持模型的范围也在不断拓展，涵盖了包括 LLM，多模态大模型，AIGC 文生图等多个种类超过 3000 个模型。
  • 简单拖拉拽，轻松创建 Agent 应用和 RAG 应用
  • 新增差异种子功能，让用户能够探索更多样的图像结果
