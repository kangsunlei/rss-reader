DeepSeek 的横空出世早有“先兆”

2/14/2025, 7:57:38 AM

📌 一句话摘要

访谈 PingCAP 和宜创科技专家，探讨 DeepSeek 突破、文生视频及 AI Agent 等 AI 热点与趋势。

📝 详细摘要

InfoQ 中文对 PingCAP 副总裁刘松及宜创科技创始人宜博进行访谈，探讨 AI 技术发展趋势。文章聚焦 DeepSeek 低成本高效的突破、 OpenAI 和谷歌的新模型、文生视频技术、企业如何结合 AI、Agent 技术等热点。专家分析国内外大模型技术发展水平，预测 Agent 技术将颠覆传统企业软件模式，并强调企业需要储备算力以应对爆发式应用，AI + Data 将变得越来越重要，或将在 2025 年迎来 Agent Network 的颠覆性创新和推理能力增强带来的新应用。

💡 主要观点

  1. 文生视频在短视频领域增长迅速 AI 视频能大量减少人类工作量，但目前在生产领域可控性和准确性仍有不足，2025 年可控性将是发展重点。
  2. DeepSeek 助力中国工程师以更少资源实现技术普惠和局部超越 DeepSeek 类似于模型领域的“拼多多”，为大模型发展带来了新的机遇，降低了技术门槛。
  3. Agent 技术将颠覆传统企业软件模式 Agent 通过 RAG、知识图谱等技术，为 SaaS 增加决策点和知识输入，实现更智能的流程管理和问题解决。
  4. 国内大模型厂商竞争激烈，追求效果而非利润，AI + Data 未来将更重要 随着模型逐渐成熟，数据和领域模型的完善程度将成为竞争的关键。

💬 文章金句

  • 文生视频是整个 AI 领域里收入增长最快的应用，远远比之前的聊天机器人还火热。
  • DeepSeek 就有人说这是模型领域的 “拼多多”，它意味着只要有明确的路径，中国工程师就可以用更少的时间更少的资源去达成并局部超越，从而实现技术的普惠化和低门槛。
  • 现在的 ChatBI 还是处于 1.0 的阶段，只能针对一定类型的问题给出反馈，将来能够回答所有关于企业的问题时才是客户真正需要的，2.0 版本的 ChatBI。
  • 我们今天的编程未来都会变成一个工程师指挥很多编程 Agent 的模式，这是不可逆的过程，会带来生产力的大幅提升。
  • 现在的技术环境可能也很难，ToC 有可能会爆发出超级应用，但 ToB 领域肯定是一个长周期，企业对 AI 的接纳需要更长的时间。
