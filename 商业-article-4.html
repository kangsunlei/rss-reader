马斯克发布「最强大模型」超越 DeepSeek，成本高出 200 多倍

2/18/2025, 3:24:06 PM

📌 一句话摘要

马斯克发布 Grok 3，以强大推理能力和 DeepSearch 挑战现有 AI 模型及搜索引擎格局，但成本高昂。

📝 详细摘要

文章报道了马斯克发布的最新一代大语言模型 Grok 3，以及基于 Grok 3 的 DeepSearch 搜索引擎。Grok 3 在推理能力上超越了 DeepSeek R1，在 LMSYS 盲测中排名第一。Grok 3 的成功归功于其背后庞大的数据中心和 20 万张显卡的支持，体现了 Scaling Law 的有效性。DeepSearch 则被定位为下一代搜索引擎，通过理解用户意图和交叉核实信息源来提供更准确的答案。文章还提到了 Grok 3 的语音模式和 API 的发布计划，以及马斯克对开源 Grok 2 的展望。值得注意的是，Grok 3 在演示环节中也出现了一些翻车案例，同时 SuperGrok 采用了订阅模式进行商业化尝试，其高昂的训练成本也是一个显著特点。

💡 主要观点

  1. Grok 3 推理能力领先，但训练成本高昂 Grok 3 在 LMSYS 盲测中表现出色，但其背后是 20 万张显卡的高成本投入，验证了 Scaling Law，但也引发了关于成本效益的讨论。
  2. DeepSearch 旨在革新搜索体验 DeepSearch 通过理解用户意图和交叉验证信息源，力求提供更精准的搜索结果，但实际效果还有待市场检验。
  3. Grok 3 语音模式和 API 即将推出 语音模式和 API 的推出将进一步拓展 Grok 3 的应用场景，但具体的性能和定价策略仍有待公布。

💬 文章金句

  • 「聪明得可怕」，这是马斯克预告 Grok 3 时的说法。
  • 作为诞生于 20 万张显卡的产物，Grok 3 断层级别的表现依旧证明了 Scaling Law 仍在生效。
  • Grok 3 通过设计一个子任务面板来完成这个工作，用户可以自己实时监测正在进行的任务，从而保证答案是可用的、可靠的。
  • 在这个基础上，agent 自主检索十分钟，抵掉自己摸索一小时，才是有效减少工作量。
  • 当 Grok 3 成熟稳定之后，大概几个月将会开源 Grok 2。
  • 最大的挑战是让整个模型在海量的 H100 上进行训练，并保持一致性
