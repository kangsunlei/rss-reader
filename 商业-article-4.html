腾讯推出快思考模型：API 成本只有 deepseek 不到一半

3/2/2025, 6:01:33 AM

📌 一句话摘要

腾讯“反常”地推出混元 Turbo S 快思考大模型，通过架构创新降低成本，提升响应速度，满足用户对快速反馈的需求，并在特定领域表现出色。

📝 详细摘要

在各大模型纷纷推出深度思考模型的同时，腾讯发布了混元 Turbo S 快思考大模型。该模型旨在通过快速响应能力满足用户在通用场景下的需求，并通过长短思维链融合，在保持文科类问题快思考体验的同时，提升理科推理能力。Turbo S 采用了 Hybrid-Mamba-Transformer 融合架构，降低了计算复杂度，减少了 KV-Cache 缓存占用，从而降低了部署成本。在定价上，Turbo S 具有明显优势，其 API 成本远低于 Deepseek 等竞争对手，仅为 Deepseek API 成本的 1/2-1/4。腾讯混元团队还分享了关于快思考模型的技术解析，包括模型架构和工程优化等方面。

💡 主要观点

  1. 快思考模型通过直觉快速响应用户在通用场景下的需求 针对 90% 的日常决策，快思考模型能够迅速给出简洁答案，无需深度思考，提升用户体验和效率。
  2. Hybrid-Mamba-Transformer 架构降低计算成本 该架构融合 Mamba 和 Transformer 的优势，在处理长序列时更高效，同时保留了 Transformer 在推理任务中的优势。
  3. 长短思维链融合提升模型理科推理能力 通过融合长短思维链，Turbo S 在保持速度优势的同时，提升了在数学、代码、逻辑等强推理任务上的表现。

💬 文章金句

  • 用户约 90% 的请求都可以依靠大模型的「直觉」（即快思考模型），无需深度思考就能精准简洁地给出答案
  • Turbo S 借鉴了腾讯的慢思考模型 Hunyuan T1 的数据，该模型使用一种称为长思维链合成的技术进行训练
  • 不管是模型 size 的 scaling 还是训练数据的 scaling，目前 scaling law 远未结束
  • 定价上，Turbo S 输入价格为 0.8 元 / 百万 tokens，输出价格为 2 元 / 百万 tokens，相比前代混元 Turbo 模型价格下降数倍，是 deepseek API 成本的 1/2-1/4
