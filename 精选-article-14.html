Gemma 3 发布：开发者指南

3/12/2025, 12:00:00 AM

📌 一句话摘要

Gemma 3 是谷歌最新且最强大的开源多模态模型，具有更强的多模态、长文本理解、数学、推理和聊天能力。

📝 详细摘要

Gemma 3 是谷歌最新一代的开源模型，相比之前的 Gemma 版本有了显著改进。它支持多模态输入，包括视觉语言，处理高达 128k Tokens 的上下文窗口，理解超过 140 种语言，并增强了数学、推理和聊天能力，包括结构化输出和函数调用。Gemma 3 提供四种尺寸（1B、4B、12B 和 27B），包括预训练模型和通用指令调整版本，基于 2T 到 14T Tokens 训练，具体取决于模型大小。它使用蒸馏、强化学习和模型合并等优化技术构建，并在谷歌 TPUs 上使用 JAX 框架进行训练。Gemma 3 还包括 ShieldGemma 2，这是一个 4B 图像安全分类器，用于合成和自然图像的安全审查，为 AI 安全做出贡献。

💡 主要观点

  1. Gemma 3 引入多模态支持，扩展了应用场景 通过集成视觉编码器，Gemma 3 可以处理图像和视频输入，执行图像分析，回答问题和识别对象，为开发多模态应用奠定基础。
  2. Gemma 3 通过各种训练技术增强模型性能 蒸馏、RLHF、RLMF 和 RLEF 等技术显著提高了 Gemma 3 在数学、编码和指令遵循方面的能力，因而在 LMArena 中获得高分。
  3. Gemma 3 提供多种部署选项，方便开发者使用 Gemma 3 支持多种部署方法，包括谷歌 GenAI API、Vertex AI 和 Cloud Run，并与 Hugging Face Transformers 和 Ollama 等工具集成，使开发人员可以轻松上手。
  4. Gemma 3 支持超过 140 种语言，扩大了其全球影响力 凭借增强的多语言能力，Gemma 3 可以应用于不同的语言环境，使其成为全球应用的强大工具。

💬 文章金句

  • Gemma 模型已被下载超过 1 亿次，社区为各种用例创建了超过 60，000 个变体。
  • Gemma 3 引入了多模态，支持视觉语言输入和文本输出。
  • ShieldGemma 2 是基于 Gemma 3 构建的 4B 图像安全分类器。
  • 我们一直对 Gemma 社区的独创性和 Gemmaverse 的爆炸性增长感到惊讶
