全球首个混合推理模型：Claude 3.7 Sonnet 来袭，真实编码力压一切对手

2/25/2025, 2:22:45 AM

📌 一句话摘要

Anthropic 发布混合推理模型 Claude 3.7 Sonnet 及智能编码工具 Claude Code，主打编码能力。

📝 详细摘要

Anthropic 发布了新型旗舰模型 Claude 3.7 Sonnet，这是一个混合推理模型，支持标准和扩展思考模式，并允许 API 用户控制思考时间。同时，Anthropic 还推出了智能编码工具 Claude Code，作为有限的研究预览版，旨在提升开发者的编码效率。Claude 3.7 Sonnet 在编码能力方面表现出色，在 SWE-bench Verified 上实现了 SOTA 性能，远远超过其他模型，尤其是在解决 GitHub 上真实软件问题方面。该模型现已通过所有 Claude 计划以及 Anthropic API、Amazon Bedrock 和 Google Cloud Vertex AI 提供。Anthropic 强调了负责任的 AI 构建，并对模型进行了广泛的测试和评估，以确保其安全性和可靠性。未来，Anthropic 希望 Claude 能够成为独立自主工作的专家级智能体，解决更具挑战性的难题。

💡 主要观点

  1. Claude 3.7 Sonnet 是首个混合推理模型，支持标准和扩展思考模式。 用户可根据需求选择快速响应或深度思考模式，并通过 API 控制思考预算，在速度、成本和质量间权衡。
  2. Claude 3.7 Sonnet 在真实软件问题解决能力上实现了 SOTA 性能。 在 SWE-bench Verified 基准测试中，Claude 3.7 Sonnet 远超 Claude 3.5 Sonnet、OpenAI 的 o3-mini (high) 和 o1 以及 DeepSeek R1。
  3. Claude Code 智能编码工具可有效提升开发者效率。 Claude Code 能够搜索代码、编辑文件、编写测试、提交代码等，一次性完成原本需要 45 分钟以上的手动任务。
  4. Anthropic 强调负责任的 AI 构建，关注模型安全与可靠性。 通过广泛测试评估，减少不必要的拒绝，并详细分析模型卡中的安全风险，训练 Claude 抵御攻击。
