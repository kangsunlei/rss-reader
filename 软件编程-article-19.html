📝 详细摘要

本文详细介绍了 xAI 公司最新推出的旗舰 AI 模型 Grok-3，该模型旨在对标 OpenAI 的 GPT-4o 和 Google 的 Gemini。文章首先概述了 Grok-3 的背景和特点，包括其与特斯拉的整合计划和“无过滤”的交互体验。随后，文章深入探讨了 Grok-3 的性能评估，强调其在多个 AI 基准测试中超越了 GPT-4o。此外，文章还介绍了 Grok-3 的新功能，如 DeepSearch 和语音模式，以及 xAI 的开源计划。最后，文章提供了 xAI API 的使用方法和白嫖攻略，明确指出通过数据共享，符合条件的团队每月可获得 $150 的免费 API 额度，并提醒了资格要求、国家/地区限制以及支付步骤等注意事项。同时推荐了一些开源项目（Open WebUI、Dify）和 AI 代码编辑器（Cursor、Windsurf、Cline），方便用户快速体验或实现高级定制。

💡 主要观点

  1. Grok-3 在多个 AI 基准测试中超越 GPT-4o。 Grok-3 在 AIME、GPQA 等关键 AI 基准测试中表现出色，特别是在数学和推理能力方面，展示了其强大的性能。
  2. xAI 计划开源 Grok 2，并提供 API 白嫖机会。 xAI 承诺在新版本完全推出后开源上一代模型，并向符合条件的团队提供每月 $150 的免费 API 额度，降低了 AI 技术的使用门槛。
  3. Grok-3 引入 DeepSearch，可扫描互联网和 X 平台提取信息，提升信息检索效率。 DeepSearch 能够扫描互联网和 X 平台提取信息并生成摘要，提升用户的信息检索效率。
