.NET MAUI 社区工具包：新增 .NET 9 支持和离线语音识别

2/21/2025, 12:00:00 AM

📌 一句话摘要

.NET MAUI 社区工具包更新，主要添加 .NET 9 支持和离线语音识别。

📝 详细摘要

微软发布 .NET MAUI 社区工具包更新，添加了 .NET 9 支持和离线语音识别。10.0.0 版本包含破坏性变更，例如不再支持 .NET 8，并要求手动分配 NMCT 行为的绑定上下文。11.0.0 版本将离线语音识别引入 `SpeechToText` API。该工具包旨在通过社区贡献增强 MAUI 开发，其贡献可能被纳入微软官方库。

💡 主要观点

  1. .NET MAUI 社区工具包旨在通过社区贡献来增强 MAUI 开发，并且这些贡献有望被纳入官方库。 该工具包允许社区贡献官方框架中所需的代码，鼓励开发者提交他们的贡献，这些贡献稍后有望被纳入官方微软库。
  2. NMCT 10.0.0 包含影响开发人员的破坏性变更。 此版本不再支持 .NET 8，需要 .NET SDK 9.0.101 和 Xcode 16.1。同时，NMCT 行为需要手动分配绑定上下文，这会影响现有项目。
  3. NMCT 11.0.0 引入了离线语音识别，增强了 MAUI 应用的功能。 此版本将离线语音识别添加到现有的 `SpeechToText` API，并通过 `OfflineSpeechToText` 作为 `ISpeechToText` 实现，从而扩大 MAUI 应用的潜在用例。

💬 文章金句

  • MAUI 是多平台应用 UI (Multiplatform Application UI) 的首字母缩写。
  • 社区工具包作为开源软件发布，并鼓励开发人员提交他们的贡献。部分工具包新增功能后续或可被纳入微软官方库。
  • .NET MAUI 社区工具包以不同的版本发布，并经常更新。频繁更新版本是由于破坏性变更，导致主版本号增加。
