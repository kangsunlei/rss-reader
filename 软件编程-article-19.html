如何在 IDE 里使用 DeepSeek-V3 和 DeepSeek-R1 满血版模型

2/13/2025, 9:19:13 AM

📌 一句话摘要

介绍如何在 JetBrains IDEs 和 VS Code 中通过通义灵码插件使用 DeepSeek-V3 和 DeepSeek-R1 模型进行 AI 辅助编程。

📝 详细摘要

本文详细介绍了如何在 JetBrains IDEs 和 Visual Studio Code 中安装和配置通义灵码插件，以便使用 DeepSeek-V3 和 DeepSeek-R1 模型进行 AI 辅助编程。DeepSeek 模型在代码补全和问题解决方面表现出色，能精准理解需求，快速生成高质量代码片段。文章涵盖了插件的安装步骤、登录流程以及智能补全、智能问答、AI 程序员等核心功能的使用方法。此外，还介绍了如何利用 AI 程序员进行多文件代码修改、生成单元测试以及问题修复等操作，旨在帮助开发者更高效地利用 AI 工具提升编码效率。

💡 主要观点

  1. 通义灵码插件支持在主流 IDE 中便捷集成 DeepSeek 模型，提升编码效率 通过在 JetBrains IDEs 和 VS Code 中安装通义灵码插件，开发者可以直接使用 DeepSeek-V3 和 DeepSeek-R1 模型进行代码补全、智能问答等操作，从而提高编码效率。
  2. AI 程序员功能可以辅助开发者完成复杂编码任务 通义灵码的 AI 程序员具备多文件代码修改和工具使用能力，能够与开发者协同完成需求实现、问题修复、单元测试生成等任务。
  3. 智能问答功能提供多种模式以解决研发问题 通义灵码的智能问答支持自由问答、代码问答、本地工程问答、Terminal 问答等多种模式，帮助开发者快速获取答案和解决思路。

💬 文章金句

  • 通义灵码 AI 程序员，具备多文件代码修改（Multi-file Edit）和工具使用（Tool-use）的能力，可以与开发者协同完成编码任务，如需求实现、问题解决、单元测试用例生成、批量代码修改等。
  • AI 程序员可以帮助开发者快速完成一个研发任务的代码编写工作，在与 AI 程序员对话过程中，可以遵循以下几点建议：清晰的需求描述、指定需要的上下文、明确生成要求、善用快照功能、审查和修改变更。
  • 单元测试智能体是 AI 程序员所具备的一种专项能力，可以针对代码变更（#codeChanges）、单个或多个代码文件批量生成单元测试文件。
