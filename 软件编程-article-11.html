反转了！微软就删除 900w 下载量的 VS Code 插件道歉，插件是安全的，误判了...

3/14/2025, 6:43:56 AM

📌 一句话摘要

微软就误删 VS Code 热门插件一事反转道歉，确认插件安全并已恢复，并承诺改进安全措施。

📝 详细摘要

文章报道了微软 VS Code 市场团队就 Material Theme 和 Material Theme Icons 插件误判事件发布的声明。此前，这两个累计下载超过 900 万的插件因被错误地检测出存在恶意代码而下架。微软承认这是一次误判，并向开发者和用户道歉。同时，微软承诺将澄清混淆代码政策，更新扫描器和调查流程，以减少类似事件再次发生，目前这两个插件已恢复使用。

💡 主要观点

  1. VS Code 插件 Material Theme 被误判为包含恶意代码，后被证实为误判 该插件拥有超过 900 万的下载量，此次误判事件影响较大，引起了开发者社区的广泛关注。
  2. 微软官方承认误判并道歉，确认插件是安全的 微软发布声明，承认错误地下架了插件，并对开发者和用户造成的不便表示歉意。
  3. 微软将改进安全策略和扫描流程，避免类似事件再次发生 微软承诺将澄清混淆代码的政策，并更新扫描器和调查流程，提升安全检测的准确性。

💬 文章金句

  • VS Code 市场团队发布了一项声明，指出由 Equinusocio 开发的这两个插件的发布者账户被错误地标识为包含恶意软件。
  • 实际上，这是 VS Code 官方的一次误判，导致开发者客户端的这两个插件被不当删除。
  • 他们对 VS Code 生态系统的安全性非常重视，并表示会从这次事件中吸取教训。
