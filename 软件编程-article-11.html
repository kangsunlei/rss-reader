Spring 新闻汇总：Framework、Data、Security、Integration 和 Modulith 发布里程碑版本

2/17/2025, 7:13:55 AM

📌 一句话摘要

Spring 生态系统多个项目里程碑版本发布汇总，包含 Spring Boot、Framework、Data、Security 等项目的更新。

📝 详细摘要

本文汇总报道了 Spring 生态系统近期多个项目的里程碑版本发布，涵盖 Spring Boot、Spring Framework、Spring Data、Spring Security、Spring Integration、Spring Modulith、Spring AI、Spring AMQP、Spring for Apache Kafka 和 Spring for Apache Pulsar 等项目。其中，Spring Data 2025.1.0 版本将 Jakarta EE 11 和 Kotlin 2.x 作为最低基线要求，Spring AI 项目则发布了 0.6.0 版本，提供了 MCP 客户端与服务器之间的新协议版本协商。这些更新旨在提升框架的稳定性、性能和功能，并为开发者提供更现代化的开发体验，例如支持新的技术栈和提高开发效率。

💡 主要观点

  1. Spring 生态系统持续更新，多个项目朝着 Jakarta EE 11 和 Kotlin 2.x 方向发展 Spring Data 等项目的更新与 Jakarta EE 和 Kotlin 的最新发展保持同步，为开发者提供更现代化的开发体验。
  2. Spring 生态系统拥抱云原生，Spring AI 项目持续演进 Spring AI 项目的更新，体现了 Spring 生态系统在云原生应用开发领域的积极探索和投入。
  3. 每个里程碑版本都包含缺陷修复、依赖升级和新特性 这些更新旨在提升框架的稳定性、性能和功能，开发者可以根据项目需求选择合适的版本升级。

💬 文章金句

  • Spring 团队还宣布，他们开始将 Spring 项目的里程碑和候选版本发布到 Maven 中央仓库，并计划在 2025 年 11 月发布 GA 版本。
  • Spring Data 团队预计在 2025 年 11 月发布 GA 版本。
  • 该团队宣布，Apache Kafka 4.0.0 的开发工作将会开始。它将基于即将发布的 Kafka Client 4.0.0 和 Spring Framework 7.0.0，并兼容 Spring Boot 4.0.0。
  • MCP 客户端与服务器之间的新协议版本协商
