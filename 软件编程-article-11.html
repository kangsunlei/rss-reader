Databricks SQL 2025 年 2 月更新速递

2/21/2025, 7:00:00 PM

📌 一句话摘要

Databricks SQL 在 AI/BI、智能体验和预测优化等方面迎来多项更新，性能显著提升，并再次荣获 Gartner 领导者称号。

📝 详细摘要

本文总结了 Databricks SQL 在过去三个月中的关键更新。Databricks 收购 BladeBridge，加速传统数据仓库向 Databricks SQL 的迁移，BladeBridge 提供代码转译、大语言模型驱动的转换以及迁移系统验证能力。AI/BI 方面，Genie 和 Dashboards 均新增功能，包括 AI 辅助生成图表、可编辑的可视化，并提升了问题评估的准确性。SQL 编辑器集成 Git 支持，并新增多结果语句等特性。此外，预测优化功能通过 AI 自动管理性能，简化数据维护，平均性能提升 22%。最后，Databricks SQL 性能显著提升，并在 Gartner® Magic Quadrant™ 中再次被评为领导者。

💡 主要观点

  1. 收购 BladeBridge 加速数据仓库迁移 BladeBridge 利用 AI 技术，加速 Oracle、SQL Server 等传统数据仓库向 Databricks SQL 的迁移。
  2. AI/BI 功能增强提升分析效率 Genie 和 Dashboards 引入 AI 辅助图表生成、可编辑可视化等功能，显著提升数据分析和报告效率。
  3. 预测优化简化数据维护 AI 驱动的预测优化自动管理数据统计信息，减少手动调优，从而提升性能并降低成本。
  4. SQL 编辑器通过 Git 集成和多项功能改进提升开发体验 SQL 编辑器集成 Git 支持，并提供多结果语句等特性，方便 SQL 代码的版本控制和协作开发。
  5. Databricks SQL 性能显著提升 通过持续优化，Databricks SQL 性能显著提升，BI 查询、ETL 任务和数据探索速度均得到提高。
