突发！DeepSeek 被美国质疑「偷窃」，遭 OpenAI 微软调查，论文曝突破英伟达护城河

2/7/2025, 2:13:44 PM

📌 一句话摘要

DeepSeek 遭数据窃取调查，意外揭示其突破英伟达 GPU 的底层技术。

📝 详细摘要

文章报道了 DeepSeek 被指控通过 OpenAI 的 API 非法获取数据用于模型训练，引发 OpenAI 和微软的调查，美国官员也对其展开国家安全调查。DeepSeek 否认使用 OpenAI 数据，并公布了 R1 模型的技术细节。文章还揭示了 DeepSeek V3 模型通过 PTX 优化实现性能提升的技术，可能打破英伟达在 AI 硬件领域的垄断，并可能撬动整个 AI 产业链的既有秩序。

💡 主要观点

  1. DeepSeek 被指控未经授权使用 OpenAI 数据进行模型训练。 微软和 OpenAI 发现 DeepSeek 可能通过 API 非法获取数据，违反 OpenAI 服务条款，美国官员也对其展开国家安全调查。
  2. DeepSeek V3 模型通过 PTX 优化，绕过 CUDA 实现性能提升。 通过对英伟达 GPU 低级汇编语言 PTX 进行优化，DeepSeek V3 模型在 H800 GPU 上实现了超越常规 CUDA 开发水平的性能。
  3. DeepSeek 的 PTX 优化代表了一种新的技术路径，尽管面临挑战，但潜力巨大。 PTX 优化虽然开发难度和维护成本高，但也为 AI 硬件和软件的协同优化提供了新的可能性。

💬 文章金句

  • 我们知道，来自中国的公司以及其他一些企业，始终在尝试『蒸馏』美国领先 AI 公司的模型。
  • PTX 就像是打开了游戏机的后盖，直接调整里面的各种配件和线路。虽然这样做很复杂，需要懂得很多专业知识，但是可以让游戏机跑得更快、性能更好。
  • 但在既有规则之外寻求突破，往往就是颠覆的开始，DeepSeek 这次在海内外掀起的技术浪潮或将有望撬动整个 AI 产业链的既有秩序。
  • DeepSeek 通过 PTX 优化，实现了对 GPU 底层架构的深度掌控。
