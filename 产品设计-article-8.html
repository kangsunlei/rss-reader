全球首个混合推理模型 Claude 3.7 降世！最强编程大脑暴击 DeepSeek R1

2/26/2025, 11:02:59 AM

📌 一句话摘要

Anthropic 发布首个混合推理模型 Claude 3.7 Sonnet，显著提升编码与推理能力，并推出智能体编程工具 Claude Code。

📝 详细摘要

文章介绍了 Anthropic 发布的新一代大语言模型 Claude 3.7 Sonnet，它是首个混合推理模型，具备快速响应和可扩展的逐步思考能力。相较于上一代模型，Claude 3.7 Sonnet 在数学、编码等方面的能力大幅提升，并在 SWE-bench 等基准测试中，超越了 DeepSeek R1 等模型，并在代码生成和 bug 修复等方面表现出色。此外，Anthropic 还推出了智能体编程工具 Claude Code，可以与开发者协作完成编程任务。Claude 3.7 Sonnet 还具备“行为扩展”的新特性，使其能够在游戏等任务中取得显著进展。该模型目前已在 Claude.ai 平台上线，用户可以免费体验。

💡 主要观点

  1. Claude 3.7 Sonnet 是首个混合推理模型，在编码和推理方面表现出色 该模型既能快速响应，也能进行逐步思考，通过 API 用户可以控制模型的思考时间，在速度和质量间进行权衡。
  2. Claude Code 是 Anthropic 推出的首个智能体编码工具，可与开发者协作 Claude Code 能够搜索和阅读代码、编辑文件、编写和运行测试，并提交和推送代码至 GitHub，成为开发者的强大助手。
  3. Claude 3.7 Sonnet 具备“行为扩展”特性，在多模态 AI 智能体能力评估中表现出色 该特性使其能够迭代调用函数、响应环境变化，并持续操作直到完成开放式任务，在游戏等任务中取得显著进展。

💬 文章金句

  • Anthropic 祭出首个混合推理 Claude 3.7 Sonnet，堪称扩展思考模式的最强模型。
