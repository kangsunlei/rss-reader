DeepSeek R1 之后，基础模型要如何做商业化？

3/3/2025, 10:30:27 AM

📌 一句话摘要

探讨了基础模型商业化的四种路径：应用订阅、MaaS API 收费、开源商用 License、以及 ToB 服务结合，并分析了各自的优劣势及代表厂商。

📝 详细摘要

文章分析了基础模型商业化的四种主要路径。第一种是向应用方向发展，通过产品订阅收费，用户粘性较差，依赖推理成本降低，以 OpenAI 的 ChatGPT 为代表。第二种是向 MaaS 方向发展，通过 API 收费，回报上限高，但面临价格战和同质化竞争，Anthropic 是代表。第三种是向基础设施方向发展，开源并通过商用 license 收费，Deepseek 和 Qwen 采取宽松的开源协议，而 Meta 和 Mistral 则在商用上有所限制。第四种是将模型与 ToB 服务结合，通过增值服务创收，但 scalability 有限，埃森哲和 Palantir 是代表。文章认为，最佳商业化策略是在资源限制下，能让模型能力演进更快、构建更丰富的技术生态、让更多用户使用的策略。

💡 主要观点

  1. 应用订阅模式用户粘性差，但依赖推理成本降低实现盈利。 通过构建通用界面解决简单任务，模型到用户价值路径短，但用户粘性是短板，需要依赖推理成本降低才能支撑订阅模式。
  2. MaaS 模式回报上限高，但面临价格战和同质化竞争。 通过 API 服务开发者，覆盖广泛用户，但基础模型易被商品化，需要在市场中应对价格战和同质化竞争。
  3. 开源商用 License 模式在 MIT / Apache 等宽松协议下难以盈利，而收紧 License 可能限制模型推广。 通过对商用开源模型收费，可以在生态中创造收入，但需要考虑对开源精神的影响，以及是否会限制模型的推广和使用。
  4. ToB 服务模式难以规模化，但可以为基础模型公司带来收入。 通过提供咨询、原厂服务和私有化部署等服务，可以获得收入，但人力成本高，难以规模化，更多是服务商的商业模式。

💬 文章金句

  • 基础模型注定被商品化，在商品的市场里须面对价格战、同质化竞争。
  • 致力于实现 AGI 的 reseach lab 是否要适当商业化，有钱招人买卡才能加速靠近 AGI？
  • 在 GenAI 发展的早期，商业化结果显然不应是基础模型公司的 top priority。
  • 颠覆性的技术面前，站在浪潮的最前沿才是最佳策略。
