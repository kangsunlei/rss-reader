Java 新闻综述：LangChain4j 1.0-Beta1、JHipster Lite、GlassFish

2/27/2025, 5:17:03 AM

📌 一句话摘要

本周 Java 生态系统多个项目迎来版本更新，涉及 LangChain4j、JHipster、GlassFish 等。

📝 详细摘要

本篇文章是对 2025 年 2 月 3 日当周 Java 领域内重要项目更新的综述。内容涵盖了 LangChain4j 1.0-Beta1 的发布，该版本对 ChatLanguageModel 接口进行了重要更新，并改进了工具执行机制。JHipster Lite 发布了 1.27.0 和 1.28.0 版本，修复了关键安全漏洞。GlassFish 发布了 7.0.22 版本，带来了漏洞修复和依赖项升级。GraalVM 发布了 Native Build Tools 0.10.5 版本，增强了对 reachability-metadata.json 文件的支持。Spring Cloud Stream Applications 发布了 2025.0.0 和 2024.0.1 版本，升级了 Spring Boot 和 Spring Cloud 依赖，并支持 Micrometer Tracing。Micronaut Framework 发布了 4.7.5 版本，包含 Micronaut Core 4.7.13，并修复了多个模块的漏洞。Quarkus 发布了 3.18.2 版本，解决了 Hibernate ORM 连接问题。WildFly 发布了 35.0.1 版本，进行了依赖项升级并修复了一些重要问题。JDKUpdater 发布了 14.0.73+109 版本，提供了 CVE 信息跟踪功能。Keycloak 发布了 26.1.1 版本，新增了 X.509 认证和密码重置邮件的配置选项。Jox 发布了 0.4.0 版本，重构了 Channel 类并优化了通道缓冲区大小的设置。这些更新和修复提升了 Java 开发的稳定性和安全性，为开发者带来了更好的体验。

💡 主要观点

  1. LangChain4j 1.0-Beta1 引入新的 Chat API 通过实验性的 chat() 方法替代 generate() 方法，为未来的 Chat API 奠定基础，并改进工具执行的策略。
  2. JHipster Lite 修复 Vitest 中的远程代码执行漏洞 升级 Vitest 版本至 3.0.5，修复 CVE-2025-24964 安全漏洞，防止跨站 WebSocket 劫持攻击。
  3. Spring Cloud Stream Applications 升级依赖并支持 Micrometer Tracing Spring Cloud Stream Applications 发布 2025.0.0 和 2024.0.1 版本，分别将依赖项升级到 Spring Boot 3.4.2/Spring Cloud 2024.0.0 和 Spring Boot 3.3.8/Spring Cloud 2023.0.5，同时支持 Micrometer Tracing。
  4. 多个 Java 框架和工具进行了依赖升级和问题修复 包括 GlassFish、GraalVM、Micronaut、Quarkus、WildFly 等，提升了稳定性和安全性。

💬 文章金句

  • LangChain4j 1.0.0 的首个测试版包含许多重大变更，例如弃用了 ChatLanguageModel 接口中定义的重载 generate() 方法，转而使用实验性的重载 chat() 方法，该方法最终将成为新 Chat API 的一部分。
  • JHipster Lite 的 1.28.0 和 1.27.0 版本进行了多项依赖项升级，其中最值得注意的是将 Vitest 升级到 3.0.5 版本，以修复关键安全问题 CVE-2025-24964。
  • JDKUpdater 的 14.0.73+109 版本于上周发布，这是一款能帮助开发者跟踪 OpenJDK 和 GraalVM 构建更新情况的工具。
