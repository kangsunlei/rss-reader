小米大模型团队 2 篇论文被 NAACL 2025 录用

2/10/2025, 1:48:13 AM

📌 一句话摘要

小米大模型团队两篇关于 GUI Agent 和机器翻译的论文被 NAACL 2025 录用，相关技术已应用于小爱翻译，提升用户体验。

📝 详细摘要

小米大模型团队在 NAACL 2025 上发表了两篇论文，分别聚焦 GUI Agent 和机器翻译。在 GUI Agent 方向，提出了 ReachAgent 模型，通过将任务拆解为页面访问和操作子任务，提高了任务完成的准确率。在机器翻译方向，提出了 PFMS 训练策略，提升了翻译质量并基于 Gemma2 基座模型训练并开源了 GemmaX2-28 系列模型，翻译性能媲美 GPT-4 Turbo 和 Google Translate。这些技术已应用于小爱翻译，提升了用户体验。小米将继续深耕底层技术，推动多语言翻译技术的发展。

💡 主要观点

  1. ReachAgent 通过拆解任务提升 GUI Agent 任务完成能力 将手机控制任务拆解为页面到达和页面操作子任务，让 Agent 更关注整体任务完成能力，而非单步最优解，实验结果表明有效提升了准确率。
  2. PFMS 策略提升开源大模型翻译性能 优先使用平行数据，在中低资源语言平行数据不足时采用单语数据补充，基于此策略训练的 GemmaX2-28 模型翻译性能优秀。
  3. 多语言翻译技术赋能小爱翻译，打破语言壁垒 小米自研的多语言翻译模型应用于小爱翻译，实现边说边译、边听边译、边看边译，提升用户在观看外语视频、浏览外文网页等场景下的体验。

💬 文章金句

  • 这是小米大模型部分研究成果的阶段性展示，同时也是践行小米科技战略中**"深耕底层技术、长期持续投入"**的又一例证。
  • 具体而言，对于每种语言的预训练数据，我们优先选择平行数据；在中低资源语言的平行数据不足时，我们采用单语数据进行补充。
  • 未来，我们将持续探索和推动多语言翻译技术发展，服务于产品应用，让全球每个人都能享受没有语言障碍的美好生活。
