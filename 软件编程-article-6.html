📝 详细摘要

本文详细介绍了 Java Development Kit (JDK) 24 和 JDK 25 的最新进展。JDK 24 带来了 24 项以 JEP（Java 增强提案）形式呈现的新特性，涵盖核心 Java 库、Java 语言规范、安全库、HotSpot 和 Java 工具五大类。文章深入探讨了其中一些关键特性，例如 Amber 项目中旨在简化初学者入门的简单源文件和实例主方法，Loom 项目中用于线程间共享不可变数据的作用域值，Panama 项目中用于优化数值计算的向量 API，以及 Leyden 项目中旨在缩短启动时间的提前类加载和链接。此外，文章还展望了 JDK 25 可能包含的新特性，包括稳定值、结构化并发、CDS 对象流、PEM API 和 HTTP/3 支持。这些改进对于提升 Java 平台的性能、安全性和开发效率具有重要意义。文章还介绍了作者 Michael Redlich 在 Java 社区的贡献。

💡 主要观点

  1. JDK 24 包含 24 项新特性，涉及核心 Java 库、Java 语言规范、安全库、HotSpot 和 Java 工具等领域。 这些新特性旨在改进 Java 语言、核心库、安全性和性能，为开发者提供更强大的工具和更高效的平台。
  2. JDK 25 预计将引入稳定值、结构化并发等多项改进。 这些改进将进一步提升 Java 的并发编程能力和性能，并简化开发流程。
  3. 多个 Java 项目（如 Amber、Loom、Panama、Leyden）为 JDK 的演进做出了重要贡献。 这些项目孵化了一系列组件，最终通过精心策划的合并纳入 JDK，不断推动 Java 技术的发展。

  • JDK 24 是自 JDK 21 以来的第三个非长期支持版本。
  • 该特性旨在'改进 Java 语言，让初学者编写第一个程序时，无需理解为大型程序设计的语言特性'。
