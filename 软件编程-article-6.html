英国旅游局：提取游客情感的及时洞察

2/18/2025, 8:00:00 PM

📌 一句话摘要

英国旅游局利用 Databricks 和 LLM 快速分析游客反馈，优化旅游服务和营销策略。

📝 详细摘要

英国旅游局 (VisitBritain) 与 Redshift 合作，利用 Databricks Mosaic AI 平台，采用 LLM (如 OpenAI GPT-4) 分析游客的自由文本反馈，以提升市场营销效果和服务质量。该方案通过翻译、过滤低质量回复后，利用 LLM 进行情感、情绪和主题分类，并将原本数天甚至数周的分析时间缩短到几秒，快速获得数据质量评分。结果整合到 Databricks Medallion 架构中。通过 LLM 作为评估者的方法评估模型输出，并根据评估结果优化原始数据。该方案帮助 VisitBritain 快速获得数据洞察，优化广告信息，并提升了问卷调查的有效性，同时，其他团队也能够复用该方案评估其他调查的反馈。

💡 主要观点

  1. 利用 LLM 能够快速分析游客反馈，提取有价值的见解。 传统人工分析耗时且主观，而 LLM 能够自动化处理大量自由文本数据，快速识别情感、主题和关键信息，为市场决策提供及时支持。
  2. 使用 LLM 作为评估者的方法可用于评估模型输出质量。 通过训练数据集和测试数据集，将 LLM 的输出与真值进行比较，可以有效评估模型的准确性和一致性，并发现数据中的潜在问题。
  3. Databricks 平台为 LLM 应用提供了强大且灵活的基础设施。 Databricks 的模块化和开源方法简化了开发和评估过程，同时 Unity Catalog 和 MLflow 等工具支持数据管理、权限控制和模型比较。
