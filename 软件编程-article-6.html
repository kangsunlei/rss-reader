微软亮相英伟达 GTC：引领人工智能创新

2/25/2025, 4:00:00 PM

📌 一句话摘要

微软将在英伟达 GTC 大会上重点展示 Azure 人工智能平台，加速企业人工智能创新。

📝 详细摘要

微软将参加英伟达 GTC 人工智能大会，展示 Azure 人工智能平台如何助力各种规模的企业构建和扩展人工智能解决方案。Azure 人工智能提供模型、服务和工具，包括大型语言模型和大型多模态模型，可通过 Azure OpenAI 服务访问。Azure 人工智能工厂通过安全和评估控制，简化人工智能应用的开发流程。该平台还提供高性能人工智能基础设施，并通过免费的人工智能服务和 Azure 额度支持初创企业。微软将通过讲座、小组讨论和展位互动，深度展示 Azure 人工智能和英伟达人工智能的最新技术。

💡 主要观点

  1. Azure 人工智能平台通过全面的模型、服务和工具，加速企业人工智能解决方案的创新与落地。 Azure 人工智能提供全面的模型、服务和工具，降低了人工智能应用程序的复杂性并加速了人工智能创新。
  2. Azure 人工智能工厂通过内置安全性和可配置的安全过滤器，简化人工智能应用程序的开发和管理。 开发人员可以使用 Azure 人工智能工厂设计、定制和管理人工智能应用程序，该工厂提供全面的人工智能运维工具和可配置的安全过滤器。
  3. 微软通过提供免费人工智能服务和 Azure 额度，赋能初创企业快速发展。 通过微软初创企业扶持计划，初创企业可以获得免费的人工智能服务、Azure 额度和专家指导。
  4. Azure 人工智能基础设施借助 NVIDIA 加速计算，提供高性能计算能力，全面支持人工智能创新。 Azure 人工智能基础设施从芯片到系统进行了优化，利用英伟达加速计算为生成式人工智能和工业应用提供峰值性能。
