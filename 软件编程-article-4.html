GitHub Copilot 启用 Function Calling 功能

2/20/2025, 6:00:52 PM

📌 一句话摘要

GitHub Copilot 引入 Function Calling 功能，通过 GPT-4o 和 Claude Sonnet 自动检索代码和工具，提升代码理解和开发效率。

📝 详细摘要

GitHub Copilot 推出 Function Calling 功能，旨在提升其理解用户意图和辅助编码的效率。该功能通过 GPT-4o 和 Claude Sonnet 等模型，能够自动搜索并利用当前文件、打开文件、整个代码库、调试器信息以及 Visual Studio 本身等相关代码或工具，从而更准确地响应用户的自然语言提问，**显著减少人工干预，提升编码效率**。未来还将集成 Web 功能，支持跨本地代码和外部数据源的查询。用户可以通过 Retry 按钮调整结果，或使用 # 引用手动指定上下文。GitHub 鼓励用户分享使用反馈，以进一步改进 Copilot 的功能和体验。

💡 主要观点

  1. Function Calling 助力 Copilot 自动检索相关代码和工具。 Copilot 可以根据用户提问，自动搜索并利用当前文件、打开文件、整个代码库等信息，无需手动指定上下文。
  2. 支持 GPT-4o 和 Claude Sonnet 模型，提升理解能力。 Function Calling 功能与 GPT-4o 和 Claude Sonnet 模型集成，提高了 Copilot 对用户意图的理解和响应准确性。
  3. 用户可以通过多种方式微调和控制 Copilot 的行为。 用户可以使用 Retry 按钮重新生成响应，或使用 # 引用手动指定文件、方法或解决方案，实现更精细的控制。

💬 文章金句

  • Function Calling 功能来了！这项重大更新将改变您与 Copilot 的交互方式。
  • 现在，Copilot 可以自动搜索更多相关代码或工具，助您解决问题。
  • Web 集成即将上线，支持跨本地代码和外部数据源提问。
