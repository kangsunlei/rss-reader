Docker Hub 策略更新：聚焦开发者体验

2/21/2025, 9:41:54 PM

📌 一句话摘要

Docker Hub 调整策略，取消镜像拉取费用并延迟存储收费，以优化开发者体验。

📝 详细摘要

Docker Hub 宣布对其策略进行重要更新，以优先考虑开发者体验。最显著的变化是完全取消了镜像拉取费用。此外，所有付费 Docker 订阅者从 2025 年 4 月 1 日起将拥有无限的镜像拉取（具有合理使用限制）。对于免费用户，Docker Hub 将已验证用户的拉取速率提升至每小时 100 次，未验证用户仍为每小时 10 次。Docker Hub 还决定无限期推迟任何存储费用，而是专注于提供新工具来帮助用户管理存储使用情况。如果未来引入存储费用，将提前六个月通知。这些变化旨在支持更多开发者、团队和企业扩展，并加强 Docker Hub 作为云原生生态系统基础的地位。

💡 主要观点

  1. 取消镜像拉取费用，降低开发者使用 Docker Hub 的成本。 取消拉取费用消除了开发者使用 Docker Hub 的一个重要障碍，使他们能够更轻松地构建、共享和协作。
  2. 提高免费用户的拉取速率限制，改善免费用户的体验。 将免费验证用户的拉取速率提高到每小时 100 次，确保了更流畅的开发体验。
  3. 提供新工具来帮助用户管理存储使用情况，并无限期推迟存储费用。 推迟存储费用让开发者可以专注于构建和共享应用程序，而不必担心存储成本。同时，Docker Hub 将提供新工具来帮助用户管理存储。

💬 文章金句

  • Docker 致力于让 Docker Hub 成为开发者、工程和运维团队构建、共享和协作的首选平台。
  • 取消拉取次数限制和费用——我们已全面取消镜像拉取收费。
  • 我们正在进行这些更改，以支持更多开发者、团队和企业扩展，从而加强 Docker Hub 作为云原生世界的基础。
