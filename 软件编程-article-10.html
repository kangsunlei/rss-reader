Kibana 新版开发工具控制台：更现代、更高效

2/13/2025, 2:18:37 PM

📌 一句话摘要

Kibana 8.16 版本升级开发工具控制台，核心在于采用 Monaco 编辑器并优化用户界面，提供更直观、高效的开发体验。

📝 详细摘要

Elasticsearch 的 Kibana 8.16 版本对开发工具控制台进行了重大升级，核心变化是用 Monaco 编辑器替换 Ace 编辑器，带来更现代化的编辑体验。同时，改进了用户界面，使其更直观易用；增加了多语言支持，允许用户复制 cURL、JavaScript 和 Python 等多种格式的请求；重新设计了工具栏，增加了导出和导入请求等功能；简化了配置过程，方便用户调整控制台设置；改进了历史记录功能，支持直接添加和运行请求；增强了响应式设计，自动适应不同屏幕尺寸。这些改进旨在提高开发效率和用户满意度。

💡 主要观点

  1. Monaco 编辑器替代 Ace 编辑器，提供更现代化的开发体验 Monaco 编辑器是 VS Code 的核心，具有更好的性能和维护性，保证 Kibana 平台编辑体验的一致性。
  2. 多语言支持提升了与各种开发环境的集成能力 控制台现在支持复制 cURL、JavaScript 和 Python 格式的请求，极大地方便了开发者在不同环境下的集成和使用。
  3. 重新设计的工具栏、简化的配置和改进的历史记录功能，减少了操作步骤，并通过响应式设计提升了在不同设备上的使用体验 用户界面和功能改进旨在提高开发效率和用户满意度。

💬 文章金句

  • Kibana 开发工具控制台一直是与 Elasticsearch 交互的开发者和运维人员的重要工具。
  • 本版本最显著的变化之一是将控制台的输入和输出编辑器从 Ace 编辑器迁移到 Monaco 编辑器。
  • 现在，控制台可以通过在水平和垂直布局之间切换来动态适应较小的屏幕尺寸。
