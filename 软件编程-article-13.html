谷歌投资 1000 万欧元，促进欧洲青少年安全与福祉

3/4/2025, 12:31:41 PM

📌 一句话摘要

谷歌投资并推出新功能，致力于提升欧洲青少年在线安全与福祉。

📝 详细摘要

谷歌宣布 Google.org 投入 1000 万欧元，用于支持欧洲的青少年安全和福祉项目，旨在创建更安全的在线环境，培养青少年的数字健康，并为看护者和教师提供必要的资源。此外，谷歌还推出了与 Livity 合作的“未来报告”研究计划，旨在通过 8000 名青少年的分享，塑造数字素养教育的未来，对未来数字素养教育产生潜在影响。同时，安卓 (Android) 手机、平板电脑和手表中内置了上课模式 (School Time) 功能，家庭链接 (Family Link) 也进行了更新，方便家长管理屏幕时间、隐私和内容过滤。谷歌还在年龄验证技术上投入，例如安卓 (Android) 的凭据管理器 API (Credential Manager API)，以提供有效的解决方案。

💡 主要观点

  1. 更新家庭链接 (Family Link) 和上课模式 (School Time)，强化家长控制。 新功能方便家长管理屏幕时间、隐私和内容过滤，限制孩子在校期间的手机功能。
  2. 谷歌投入 1000 万欧元支持欧洲青少年安全项目。 该资金将用于支持欧洲的社会影响组织，例如 The School with Class Foundation，旨在创建更安全的在线环境，并为青少年、看护者和教师提供资源。
  3. 推出“未来报告”研究计划，听取青少年声音。 该计划与 Livity 合作，旨在通过 8000 名青少年的分享，塑造数字素养教育的未来。

💬 文章金句

  • 谷歌安全工程中心 (GSEC) 位于都柏林，其工作基础是提供各种控制、工具和安全程序，以保护年轻用户免受网络侵害。
  • 我们认为年轻人的观点是这些努力的核心。
  • 任何年龄验证解决方案都必须优先考虑隐私，并让用户控制其个人信息。
