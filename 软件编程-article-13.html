从 DeepSeek 到 Manus：如何实现本地 LLM 微调+联网开发？

3/12/2025, 7:54:43 AM

📌 一句话摘要

本文介绍了如何使用 LLaMA-Factory 对本地 LLM，如 DeepSeek 模型，进行微调和联网功能开发，并探讨了 AI 技术在实际安全业务中的应用场景和未来趋势。

📝 详细摘要

文章详细阐述了如何利用 LLaMA-Factory 框架对本地大语言模型（LLM）进行微调，并为其添加联网功能。首先，文章介绍了模型微调的必要性，即通用大模型在特定领域存在“博而不精”的问题，因此需要进行领域适配和任务聚焦。接着，文章以 DeepSeek-R1-Distill-Qwen-1.5B 模型为例，演示了如何通过 LLaMA-Factory 进行微调，并详细讲解了环境准备、数据准备、微调原理（LoRA 微调）和模型合并等步骤。其次，文章介绍了如何为本地 LLM 添加联网功能，包括整体架构、LLM 调用、向量数据库初始化、URL 规范化、数据存储、网页爬取、爬虫合规检查和网络搜索等关键技术。最后，文章探讨了 AI 技术在实际安全业务中的应用场景，如利用代码语义理解进行 APK 病毒检测、通过页面诱导话术识别进行网址安全检测和结合沙箱动态环境监控进行 APP 隐私检测，并对 AI 技术的发展趋势进行了预测，包括垂直领域模型胜出（数据质量和专业知识是关键）、多模态成为标配和 T 型技能结构更有竞争力等。文章还对开源与商业模式进行了思考，认为开源大模型与 API 服务将长期共存，垂直领域的 AI 服务商机巨大。

💡 主要观点

  1. 通过 LLaMA-Factory 可对本地 LLM 进行高效微调 利用 LoRA 等轻量级微调方法，在特定领域数据上进行训练，提升模型在垂直领域的专业能力和准确率。
  2. 联网功能扩展了本地 LLM 的应用场景 通过网页爬取、信息检索等技术，使 LLM 能够获取最新的外部信息，从而提供更全面、实时的回答。
  3. AI 技术能提升病毒检测的准确性 通过代码语义理解和多维度关联分析，AI 技术能够更准确地识别恶意 APK 变种，提高病毒检测的效率和准确性。
  4. AI 技术能增强网址安全防护能力 通过识别页面诱导话术、综合判断多重证据和识别高仿界面，AI 技术能够有效防范钓鱼网站攻击，增强网址安全防护能力。

💬 文章金句

  • 拥有一个定制化的、能够理解你业务需求的 AI 助手不再是科幻，而是触手可及的现实。
  • 与其花大力气追赶最新的通用模型，不如考虑如何在自己熟悉的业务领域构建专业模型。
  • 在 AI 时代，纯粹的技术专家可能不如复合型人才吃香。
  • 开源模型的优势在于可控性和定制化，适合对数据安全有高要求或需要深度定制的场景；而 API 服务的优势在于使用便捷、持续更新、技术门槛低，适合快速验证和非核心业务场景。
  • 技术再强大，也只是放大人类意志的工具，而不是替代人类价值的终极答案。
