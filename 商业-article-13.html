历史分水岭：DeepSeek GitHub 星数超越 OpenAI！大佬揭秘仅用 450 美元训推理模型

2/8/2025, 4:31:50 AM

📌 一句话摘要

本文解读了 DeepSeek R1 的训练方法，强调了其在推理模型方面的创新，并探讨了低成本构建推理模型的多种开源方法，突显了开源社区在 AI 发展中的重要作用。

📝 详细摘要

文章主要围绕 DeepSeek R1 模型的训练方法及其在开源社区的影响展开，指出 DeepSeek 在 GitHub Star 数上超越了 OpenAI。文章澄清了关于 DeepSeek 的一些误解，例如是否绕过 CUDA 架构以及 R1 的训练成本。文章介绍了四种构建推理模型的方法，包括推理时扩展、纯强化学习（RL）、监督微调和强化学习（SFT + RL）以及纯监督微调（SFT）和蒸馏。特别强调了在预算有限的情况下，Sky-T1 项目仅用 450 美元就训练出与 OpenAI o1 性能大致相当的模型，证明了低成本构建推理模型的可能性。DeepSeek R1 在推理效率上更具优势，优于 OpenAI o1。

💡 主要观点

  1. DeepSeek R1 的成功得益于纯强化学习和 SFT + RL 的创新结合，以及定制 PTX 指令优化底层硬件性能。 DeepSeek R1-Zero 通过纯 RL 训练，展示了推理能力，而 DeepSeek R1 则通过结合 SFT 和 RL，进一步提升了性能，并通过定制 PTX 指令优化底层硬件性能。
  2. 模型蒸馏是低成本构建推理模型的有效途径。 DeepSeek 团队的 R1 蒸馏模型以及 Sky-T1 项目证明，通过蒸馏可以以较低的成本获得强大的推理表现。
  3. 旅程学习为推理模型研究提供了新思路。 旅程学习通过让模型从错误中学习，可能有助于加强自我纠错能力，提升推理模型的可靠性。
  4. 开源社区在推动 AI 推理模型发展中扮演关键角色，Sky-T1 和 TinyZero 等项目证明了低成本创新的可能性。 开源项目 Sky-T1 仅花费 450 美元就实现了与 OpenAI o1 相当的性能，TinyZero 则以不到 30 美元的成本展示了小模型通过纯 RL 产生推理能力的可能性。

💬 文章金句

  • DeepSeek 通过编写 PTX 解决了跨芯片通信瓶颈，降低了开销、提升了效率，为其他开发者提供了借鉴。
  • DeepSeek R1 在推理时间上更高效。
  • 模型蒸馏提供了一个更具成本效益的替代方案。
  • 旅程学习或许有助于加强自我纠错能力，提升推理模型的可靠性。
