腾讯元宝接入 Deepseek-R1 满血版，支持联网搜索并整合公众号信息源

2/16/2025, 9:54:44 AM

📌 一句话摘要

腾讯元宝接入 Deepseek-R1 满血版，增强联网搜索和微信生态信息整合能力。

📝 详细摘要

腾讯元宝 AI 助手迎来重大更新，接入 Deepseek-R1 满血版模型，用户可免费使用。该模型支持联网搜索，并整合微信公众号、视频号等腾讯生态信息源，提供更稳定、实时、全面、准确的答案。腾讯元宝依托腾讯云的算力优势，保障服务稳定性，并优化推理效率。此外，腾讯云开发也宣布支持开发者通过少量代码将 DeepSeek 大模型能力接入小程序。腾讯云还上线了 DeepSeek-R1 及 V3 原版模型 API 接口，并支持联网搜索，方便开发者快速搭建基于 DeepSeek 大模型的联网应用。

💡 主要观点

  1. 腾讯元宝接入 Deepseek-R1 提升了 AI 助手能力 Deepseek-R1 满血版模型为腾讯元宝带来更强的语言理解和生成能力，提升用户体验。
  2. 联网搜索和生态整合是腾讯元宝的差异化优势 通过联网搜索和整合微信生态信息，腾讯元宝可以提供更全面和准确的回答，区别于其他 AI 应用。
  3. 腾讯云为 DeepSeek 模型应用提供了基础设施和开发支持 腾讯云提供算力保障、API 接口和开发平台，降低了开发者使用 DeepSeek 模型的门槛。
  4. 腾讯云率先支持 DeepSeek API + 联网搜索 腾讯云成为国内首家支持 DeepSeek API 接口并提供联网搜索功能的云厂商，体现了其在该领域的领先地位。

💬 文章金句

  • 腾讯元宝提供的 DeepSeek 支持联网搜索，并整合了微信公众号、视频号等腾讯生态信息源，能为用户提供更稳定、实时、全面、准确的回答。
  • 腾讯元宝采用 DeepSeek-R1 671B 满血版，这是系列模型中能力最完整、效果最优的版本。
  • 腾讯云成为国内率先支持 DeepSeek API+ 联网搜索的云厂商。
