Google Cloud：用 LCA 衡量气候影响

3/12/2025, 12:00:00 AM

📌 一句话摘要

Google Cloud 通过生命周期评估 (LCA) 分析数据中心硬件（特别是张量处理单元 TPU）的影响，从而显著提升人工智能 (AI) 工作负载的碳效率。

📝 详细摘要

Google Cloud 致力于通过生命周期评估 (LCA) 全面了解其环境影响。LCA 是一种评估产品或服务在其整个生命周期内的环境影响的方法。Google Cloud 的碳排放核算团队开发了一种 LCA 方法，用于评估数据中心硬件供应链（包括 AI/ML 加速器、计算机器、存储平台和网络设备）相关的碳排放。Google Cloud 与供应链合作伙伴合作，通过收集一手数据来改进 LCA 数据收集方式，并参与制定一致的 LCA 指南，从而推动硬件行业的标准化。最近的一项 LCA 研究表明，通过更高效的 TPU 硬件设计，AI 工作负载的碳效率提高了 3 倍。Google Cloud 将持续改进 LCA 方法，推动可持续发展。

💡 主要观点

  1. LCA 用于衡量硬件制造过程中的碳排放，识别排放来源，并确定减少排放的具体方法。 Google Cloud 的 LCA 方法符合全球 LCA 标准，并经过 Fraunhofer IZM 的严格审查，确保其完整性、准确性和对行业标准的遵守。
  2. Google Cloud 正在与供应商合作收集一手数据，以提高 LCA 数据的准确性。 通过与供应商直接合作，Google Cloud 能够捕获整个制造过程中发生的直接流动（即与自然环境的材料和能源交易）。
  3. Google Cloud 通过引入计算碳强度 (CCI) 指标，发现 TPU 硬件设计两代之间的碳效率提升了 3 倍。 LCA 研究表明，更高效的 TPU 硬件设计显著提升了 AI 工作负载的碳效率。
