Google Cloud KMS 正式发布量子安全数字签名预览版

2/21/2025, 12:00:00 AM

📌 一句话摘要

Google Cloud KMS 发布量子安全数字签名预览版，支持 FIPS 204 / 205 标准，为未来量子计算时代的数据安全保驾护航。

📝 详细摘要

Google Cloud 发布了 Cloud KMS 的量子安全数字签名预览版，支持 FIPS 204 和 FIPS 205 标准，旨在帮助开发者提前构建和实施后量子密码学，以应对量子计算机破解现有公钥密码系统的潜在风险。Google 自 2016 年起在 Chrome 中测试 PQC，并积极采取措施，包括使用 PQC 保护内部通信，为 Google Cloud 加密产品制定了全面的 PQC 战略，彰显了其在后量子密码学领域的长期投入与领先地位。Cloud KMS 的 PQC 路线图包括支持 NIST 的 PQC 标准，并提供软件和硬件支持。此外，Google 还与 HSM 供应商和 EKM 合作伙伴合作，以实现量子安全密码学。Cloud KMS 现在提供量子安全数字签名，允许客户使用现有 API 通过存储在 Cloud KMS 中的密钥对，利用 NIST 标准化的量子安全密码学来加密签名数据和验证签名。此版本支持 ML-DSA-65 和 SLH-DSA-SHA2-128S 算法。Google 承诺将持续关注后量子密码学的发展，并根据量子密码分析领域的变化进行调整。

💡 主要观点

  1. 量子计算发展对现有密码系统构成威胁，需提前布局后量子密码学。 量子计算机有潜力破解广泛使用的公钥密码系统，因此开发者需要立即构建和实施后量子密码学，以降低潜在风险。
  2. Google Cloud KMS 推出量子安全数字签名预览版，支持 NIST 标准，彰显 Google 在量子安全领域的领先地位。 Cloud KMS 现在提供量子安全数字签名，支持 FIPS 204 和 FIPS 205 标准，帮助客户测试和集成这些签名方案到现有工作流中。
  3. 实施量子安全 DSA 是应对未来挑战的明智之举。 对于部署长期信任根或为管理关键基础设施的设备签署固件，应立即考虑针对此威胁向量的缓解方案。

💬 文章金句

  • The continued advancement of experimental quantum computing has raised concerns about the security of many of the world's widely-used public-key cryptography systems.
  • Fortunately， post-quantum cryptography (PQC) offers a way of mitigating these risks using existing hardware and software.
  • Cloud KMS now offers quantum-safe digital signatures， so customers can use our existing API to cryptographically sign data and validate signatures using NIST-standardized quantum-safe cryptography with key pairs stored in Cloud KMS.
  • The sooner we're able to secure these signatures， the more resilient the digital world's foundation of trust becomes.
