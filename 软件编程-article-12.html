Vercel 边缘请求监控：新增流量参数，洞察更深

2/24/2025, 1:00:00 PM

📌 一句话摘要

Vercel 边缘请求仪表盘新增流量参数，提升网络安全和问题排查。

📝 详细摘要

Vercel 边缘请求仪表盘已扩展，现在可以显示更多请求数据，包括用户代理、IP 地址、JA4、来源和主机名。这些数据洞察适用于所有套餐，可以帮助用户监控流量模式并识别潜在威胁，用户可以使用 Vercel 防火墙来解决潜在威胁。路由级别数据仅对 Observability Plus 用户开放。

💡 主要观点

  1. 与 Vercel 防火墙联动，实现安全闭环。 通过可观测性发现威胁后，可直接使用 Vercel 防火墙进行防御，提升整体安全性。
  2. 边缘请求仪表盘新增流量参数，提升可观测性。 通过用户代理、IP 地址等参数，更全面地了解流量来源和特征，为安全防护提供数据支持。
  3. 适用于所有套餐，降低使用门槛。 所有 Vercel 用户均可使用新增的流量监控功能，及时发现潜在的安全风险。

💬 文章金句

  • We've expanded the Edge Request dashboard in Vercel Observability to show additional request data by: User agent， IP address， JA4， Referrer， Hostname
  • Available on all plans， these insights help you monitor traffic patterns and identify potential threats， which you can address using Vercel Firewall.
