苹果 M/A 系列 CPU 曝出 FLOP/SLAP 侧信道漏洞

2/9/2025, 12:00:00 AM

📌 一句话摘要

研究人员发现苹果 M/A 系列 CPU 存在 FLOP/SLAP 侧信道漏洞,或致 Mac/iPhone/iPad 信息泄露

📝 详细摘要

佐治亚理工学院和波鸿鲁尔大学的研究人员在苹果的 M 和 A 系列 CPU 中发现了两个新的侧信道攻击漏洞,名为 FLOP 和 SLAP。这些漏洞影响了 2021 年后发布的 Macs、iPhones 和 iPads,可能允许攻击者在使用 Chrome 和 Safari 等浏览器访问恶意网站时泄露敏感信息,如电子邮件内容、浏览历史、位置信息、日历事件和信用卡信息。漏洞源于加载地址预测器(LAP)和加载值预测器(LVP)的预测错误,导致在投机执行期间可能发生数据泄露。FLOP 影响使用 M3/A17 及更新的 CPU 的设备,而 SLAP 影响使用 M2/A15 及更新的 CPU 的设备。苹果公司已收到通知,但尚未发布修复程序。这些侧信道攻击难以检测,并且利用了 CPU 的微架构。

💡 主要观点

  1. FLOP 和 SLAP 利用了 CPU 的预测机制 FLOP 和 SLAP 分别利用了 CPU 的加载地址预测器(LAP)和加载值预测器(LVP)。当这些预测器做出不正确的猜测时,CPU 可能会在投机执行期间执行错误计算,从而导致数据泄露。
  2. SLAP 是一个 Safari 特有的漏洞 由于 Safari 缺乏同源策略隔离,攻击者可以利用 SLAP 漏洞来恢复电子邮件内容和浏览历史。
  3. FLOP 影响 Chrome 和 Safari 攻击者可以利用 FLOP 漏洞,使用任意读取原语从 Safari 和 Chrome 恢复位置历史记录、日历事件和信用卡信息。
  4. 缓解措施需苹果提供 OS 级别补丁 虽然苹果尚未发布补丁,但存在可行的缓解措施,这些措施需要在操作系统层面进行修改,且只有苹果才能实施。

💬 文章金句

  • SLAP affects devices using the M2/A15 and newer CPUs with a Load Address Predictor (LAP), while FLOP affects devices using the M3/A17 and newer generations with a Load Value Predictor (LVP).
  • For SLAP, the researchers demonstrated an end-to-end attack on Safari in which an unprivileged attacker can recover email content and browsing history.
  • For FLOP, they demonstrated how an attacker can recover location history, calendar events, and credit card information from Safari and Chrome using arbitrary read primitives.
