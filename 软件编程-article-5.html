Cloudflare Radar 新增 DNS 分析功能：洞察 TXT 记录与 PTR 记录的应用

2/27/2025, 2:00:00 PM

📌 一句话摘要

Cloudflare Radar 发布新的 DNS 页面，利用 1.1.1.1 DNS 解析器数据提供互联网活动的可视化分析。

📝 详细摘要

Cloudflare Radar 扩展功能，推出新的 DNS 页面，通过分析 1.1.1.1 DNS 解析器的流量数据，提供对互联网活动的深入洞察。该页面展示了全球、地区和 ASN 层面的流量趋势、查询类型、响应代码和 TTL 值等具体数据，以及协议使用情况和 DNSSEC 使用情况。通过这些数据可以识别网络中断或 DNS 封锁事件。该页面还提供 DNSSEC 客户端感知和端到端安全性的数据，揭示了 DNS 安全领域的挑战与机遇。这些数据对于网络安全研究人员、ISP 以及关注互联网发展趋势的专业人士具有重要参考价值。

💡 主要观点

  1. Cloudflare Radar 新的 DNS 页面通过分析 DNS 流量，揭示了互联网使用模式、隐私保护趋势和安全风险。 通过分析全球、地区和 ASN 层面的流量趋势，以及协议使用情况、查询和响应特征以及 DNSSEC 使用情况，可以洞察互联网活动。
  2. DNS 传输协议的使用分布反映了用户对隐私的关注程度。 不同地区 DoT 和 DoH 的使用比例差异表明，某些地区用户对 DNS 查询的隐私保护更为重视，或与特定平台（如 Android）的 DoT/DoH 配置有关。
  3. DNS 记录类型的查询量可以间接反映相关协议的采用情况。 通过分析 A 和 AAAA 记录的查询占比，可以衡量 IPv4 和 IPv6 的普及程度，以及 HTTPS 等协议的采用情况。
  4. DNS 缓存命中率和 TTL 值的分析能够反映域名和服务的稳定性和动态性。 通过分析 DNS 缓存命中率和 TTL 值，可以了解域名和服务的稳定性和动态性，以及缓存策略的有效性。
  5. DNSSEC 客户端感知度较低，端到端安全性仍有待加强。 大多数客户端未启用 DNSSEC，导致大量 DNS 交互未受到篡改保护，需要进一步推广加密和 DNSSEC。

💬 文章金句

  • 通过对这些流量的聚合分析，我们能够获得对互联网活动的独特洞察，超越了简单的 Web 流量趋势。
  • Cloudflare Radar 在现有分析的基础上，结合域名页面的 DNS 解析器数据，推出了全新的 DNS 页面。该页面旨在更清晰地展示 1.1.1.1 DNS 解析器流量的总体趋势和使用模式。
