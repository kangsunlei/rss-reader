Visual Studio 公开预览版现已支持 GPT-4o Copilot 代码补全模型

3/11/2025, 3:00:21 PM

📌 一句话摘要

Visual Studio 发布了基于 GPT-4o 的 GitHub Copilot 代码补全模型，提供更精准的代码建议。

📝 详细摘要

Visual Studio 现已推出基于 GPT-4o 的 GitHub Copilot 代码补全模型，为开发者提供更智能快速的代码补全。该模型基于 GPT-4o mini，并在超过 30 种流行编程语言的 275，000 多个高质量公共存储库上进行了训练，显著提升了代码补全的准确性和性能。Visual Studio 用户可以在 GitHub Copilot 设置中选择“GPT-4o Copilot”来体验新模型。Copilot Pro 用户可立即使用该模型；Copilot 商业版或企业版用户需由管理员启用；Copilot Free 用户使用该模型会计入每月免费额度。Microsoft 鼓励用户分享使用体验，帮助我们改进 Visual Studio GitHub Copilot。

💡 主要观点

  1. GPT-4o Copilot 模型基于大量高质量代码仓库训练，显著提升了代码补全的准确性 该模型通过在超过 275，000 个公共存储库上进行训练，提高了代码补全的准确性和性能，从而帮助开发者更高效地编写代码。
  2. 不同 GitHub Copilot 订阅用户对 GPT-4o 模型的访问方式不同 Copilot Pro 用户可以立即使用，Copilot 商业版或企业版用户需要管理员启用，Copilot Free 用户的使用量会计入免费额度。
  3. Microsoft 鼓励用户积极提供反馈，以改进 Visual Studio GitHub Copilot 用户反馈对于优化 AI 辅助开发工具至关重要，可以帮助 Microsoft 更好地了解用户需求，提升产品质量。

💬 文章金句

  • 我们很高兴地宣布，从 17.14 Preview 2 开始，新的 GPT-4o Copilot 代码补全模型现已在 Visual Studio 中向 GitHub Copilot 用户提供！
  • 这个新模型基于 GPT-4o mini，已经在超过 30 种流行编程语言的 275，000 多个高质量公共存储库上进行了额外训练。
  • 开发者可以期待更准确的建议和更好的性能。
