MongoDB Atlas 在 Azure 和 Google Cloud 上实现安全扩展和新功能

3/6/2025, 4:57:01 PM

📌 一句话摘要

MongoDB Atlas 通过 Azure Private Link 增强了 Azure 上的安全连接；数据联邦和在线归档在 Google Cloud 上全面可用，提升了数据安全性、可扩展性和灵活性。

📝 详细摘要

MongoDB 宣布了 MongoDB Atlas 在 Azure 和 Google Cloud 上的两项重大更新：Azure Private Link 支持以及 Google Cloud 上数据联邦和在线归档的全面可用。Azure 用户现在可以通过 Azure Private Link 建立与 MongoDB Atlas 数据联邦、Atlas 在线归档和 Atlas SQL 的安全连接。这增强了安全性，减少了延迟，并提高了可扩展性。Google Cloud 用户现在可以利用全面可用的 MongoDB Atlas 数据联邦和在线归档。这使得他们能够跨多个来源查询数据，优化存储成本，并实现多云灵活性。这些更新旨在简化开发流程，提高效率，降低配置的复杂性，并增强安全性、性能和成本效益。

💡 主要观点

  1. Azure Private Link 为 MongoDB Atlas 提供安全连接。 借助 Azure Private Link，Azure 用户可以安全地连接到 MongoDB Atlas 数据联邦、Atlas 在线归档和 Atlas SQL。这避免了将数据暴露于公共互联网的风险。
  2. MongoDB Atlas 数据联邦和在线归档在 Google Cloud 上全面可用。 Google Cloud 用户现在可以利用数据联邦跨多个数据源运行查询。同时，他们可以使用在线归档将不常访问的数据移动到成本较低的存储。
  3. 优化存储成本，简化跨数据源查询。 Azure 和 Google Cloud 上的增强功能使开发人员能够构建可扩展且安全的应用程序。这有助于优化存储成本，并避免供应商锁定。

💬 文章金句

  • MongoDB 致力于简化数据驱动的强大应用程序的开发，无论数据位于何处。
  • 通过将敏感数据保留在公共互联网之外来减少安全风险。
  • 跨实时数据库、云存储和数据湖运行单个查询，而无需复杂的提取、转换和加载 (ETL) 管道。
