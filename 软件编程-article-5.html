Node.js — Node v23.9.0 (当前版本)

2/26/2025, 4:41:09 PM

📌 一句话摘要

Node.js v23.9.0 发布，新增 TLSA 记录查询和线程 CPU 使用统计功能，并更新依赖。

📝 详细摘要

Node.js 发布了 v23.9.0 版本。此版本引入了多项改进和修复，包括：DNS 模块新增 TLSA 记录查询和解析功能，增强了 HTTPS 安全性；process 模块新增 threadCpuUsage 方法，允许开发者获取线程级别的 CPU 使用情况，从而诊断多线程应用的性能瓶颈；更新了 cjs-module-lexer、ngtcp2、SQLite 和 ada 等多个依赖，以提升性能和安全性。此外，还包括构建流程改进、文档更新、内部模块优化和测试用例增强，以提升整体稳定性和开发者体验。

💡 主要观点

  1. 利用新增的 TLSA 记录查询功能，Node.js 应用可以更安全地验证 TLS 证书，增强 HTTPS 安全性。 新增的 TLSA 记录查询和解析功能，允许 Node.js 应用验证 TLS 证书，提升网络连接的安全性。
  2. process 模块新增 threadCpuUsage 方法，允许开发者获取线程级别的 CPU 使用情况，从而更精确地诊断和优化多线程 Node.js 应用的性能瓶颈。 threadCpuUsage 方法可以帮助开发者更精确地监控和优化多线程 Node.js 应用的性能。
  3. 依赖更新提升了 Node.js 的性能和安全性。 更新 cjs-module-lexer、ngtcp2、SQLite 和 ada 等依赖，修复已知问题，并引入性能优化。
