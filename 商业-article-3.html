⚡️ OpenAI 智能体平台发布

3/11/2025, 5:39:48 PM

📌 一句话摘要

OpenAI 发布智能体平台，提供 API、工具及 SDK，助力开发者构建智能体应用。

📝 详细摘要

OpenAI 发布智能体平台，旨在简化智能体应用的开发。该平台包含响应 API，是对话补全 API 的超集，内置网页搜索（由 gpt-4o-search-preview 和 gpt-4o-mini-search-preview 模型驱动，支持内联引用）、计算机使用（Operator 背后的模型）和文件搜索等工具。此外，该平台还提供开源智能体 SDK，包含智能体、移交、护栏、追踪与可观测性等关键组件，用于简化多智能体工作流的开发和调试。文件搜索工具的定价为每千次查询 2.50 美元，文件存储为每天每 GB 0.10 美元，首 GB 免费。

💡 主要观点

  1. 响应 API 集成了多种工具，是构建灵活的智能体应用的关键。 简化工具集成，支持长期任务。
  2. 网页搜索 API 提供实时的网页搜索功能。 支持内联引用，精准定位网页答案。
  3. 智能体 SDK 促进了多智能体工作流。 包含关键组件，简化多智能体应用开发。
