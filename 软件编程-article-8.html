📝 详细摘要

Node.js v20.18.3 'Iron' (LTS) 版本发布，此版本包含多项值得关注的更新。ESM 的导入属性和 JSON 模块被标记为稳定版，增强了模块系统的稳定性。依赖项方面，c-ares、zlib、corepack、simdutf、acorn 等多个依赖项进行了更新，修复了安全漏洞，提升了性能，并增强了对新硬件和操作系统的兼容性。构建方面，修复了 loongarch64 架构的 DESTCPU 设置，并改进了 GN 构建。文档方面，添加了大量 ESM 示例，并对现有文档进行了修订和完善，提升了可读性和实用性。此外，还包括一些基准测试和命令行工具的改进，例如新增了 `--heap-prof` 标志。

💡 主要观点

  1. ECMAScript 模块导入属性和 JSON 模块标记为稳定版 标志着 Node.js 在标准化和稳定化 ECMAScript 模块方面迈出了重要一步，开发者可以更放心地在生产环境中使用这些特性。
  2. 多个依赖项更新提升性能与安全性 包括 c-ares、zlib、corepack、simdutf、acorn 等关键依赖的升级，修复已知问题，引入性能优化，并增强了对新硬件和操作系统的兼容性。
  3. 构建改进增强开发体验 修复了构建过程中的一些问题，提高了构建效率和稳定性。
  4. 文档的持续完善提升可读性和实用性 对文档进行了大量更新和完善，使得开发者更容易理解和使用 Node.js。

💬 文章金句

  • esm: mark import attributes and JSON module as stable
  • cli: add `--heap-prof` flag available to `NODE_OPTIONS`
  • crypto: update root certificates to NSS 3.104
