Azure AI 代理服务公开发布预览，支持 AI Foundry SDK 和门户

2/21/2025, 12:00:00 AM

📌 一句话摘要

Azure AI 代理服务公开发布预览，赋能开发者构建 AI 代理应用。

📝 详细摘要

微软发布了 Azure AI 代理服务的公开发布预览版，该服务通过 Azure AI Foundry SDK 和门户提供，旨在帮助开发者设计、定制和部署 AI 代理，这些代理可以与微软生态系统的其余部分进行原生连接。该服务允许企业开发支持代理，通过利用 OpenAPI 定义的工具和 Azure 函数自动执行重复性任务，还集成了 OpenTelemetry 追踪和 Application Insights，以便开发者监控代理行为。Azure AI 代理服务简化了 AI 代理的开发流程，并支持通过 Semantic Kernel 或 AutoGen 的 Assistants API 处理复杂场景，它与 AWS (Amazon Bedrock 代理) 和 Google (Vertex AI 代理构建器) 提供的类似服务形成竞争，可以被视为一种代理即服务 (AaaS)。

💡 主要观点

  1. Azure AI 代理服务简化了 AI 代理的设计、定制和部署，显著降低了开发门槛，并提供了一系列便捷的开发工具，加速应用创新。 该服务提供了一个平台，用于构建、部署和管理 AI 代理作为微服务，从而减少了开发人员在计算、网络和存储方面的管理负担。
  2. 代理可以通过 OpenAPI 定义的工具和 Azure 函数自动执行重复性任务，例如发送邮件、安排会议、更新数据库、处理客户咨询等。 这使得企业能够构建支持代理，从而提高效率并减少手动工作量。
  3. 开发者可以使用 Semantic Kernel 或 AutoGen 的 Assistants API 来处理复杂场景。 这些工具支持多 Agent 的编排，从而实现更复杂的自动化工作流程。
  4. Azure AI 代理服务与微软生态系统原生连接。 这使得 AI 代理能够无缝访问并充分利用微软的各项服务与资源。

💬 文章金句

  • Azure AI Agent Service is a flexible， use-case-agnostic platform for building， deploying， and managing AI agents as micro-services.
  • Agent-as-a-Service (AaaS) redefines how businesses leverage AI， moving beyond static software to autonomous AI-driven agents.
  • Creating agents with Azure AI Agent Service is easy， either through the code-first Azure AI Foundry SDK or the Azure AI Foundry portal experience.
