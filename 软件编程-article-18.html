针对苹果 M 和 A 系列 CPU 的新型侧信道攻击

2/28/2025, 12:37:36 PM

📌 一句话摘要

针对苹果 M 和 A 系列 CPU 的新型侧信道攻击 FLOP 和 SLAP，利用 CPU 预测机制缺陷，可能导致用户在使用 Chrome 和 Safari 浏览器访问网站时泄露敏感信息，威胁严重。

📝 详细摘要

文章揭示了针对苹果 M 和 A 系列 CPU 的两种新型侧信道攻击：FLOP 和 SLAP。这两种攻击利用 CPU 微架构优化中加载地址预测器（LAP）和加载值预测器（LVP）的预测错误，可以在 2021 年及以后发布的 Mac、iPhone 和 iPad 上执行，影响使用 Chrome 和 Safari 浏览器访问热门网站的用户，可能泄露用户的浏览历史、邮件内容和信用卡信息等敏感数据。研究人员已将漏洞告知苹果公司，但苹果尚未发布补丁。FLOP 影响 Chrome 和 Safari，而 SLAP 则是 Safari 独有的，因为 Safari 在同一浏览器上下文组内未实现隔离。

💡 主要观点

  1. FLOP 和 SLAP 攻击利用 CPU 预测机制的缺陷，导致数据泄露 FLOP 利用加载值预测器（LVP），SLAP 利用加载地址预测器（LAP），在预测错误时导致数据泄露，对用户隐私构成严重威胁。
  2. 攻击可导致多种敏感信息泄露，威胁用户隐私 包括浏览历史记录、日历事件、信用卡信息、邮件内容等，攻击利用难度较低。
  3. 不同浏览器受影响程度不同 Safari 浏览器由于未在同一浏览器上下文组内实现隔离，更容易受到 SLAP 攻击，而 FLOP 影响 Chrome 和 Safari。

💬 文章金句

  • SLAP 影响使用 M2 / A15 及更新型号 CPU 的设备，这些 CPU 配备了加载地址预测器（LAP），而 FLOP 影响使用 M3 / A17 及更新型号的设备，这些 CPU 配备了加载值预测器（LVP）。
  • 由于这两种攻击都是基于微架构的，因此不会留下任何痕迹。
  • LAP 会监控来自相同加载指令的地址，用以推测性地加载预测地址，该地址可能错误地指向固定的机密信息（即从未被 CPU 读取）。
  • 随着摩尔定律逐渐失效，出现了更多独特的微架构优化，我们相信 LAP 可能不是苹果独有的，现在或在未来也可能会在其他地方出现。
