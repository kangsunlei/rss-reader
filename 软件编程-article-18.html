SAP Databricks 发布

2/13/2025, 10:30:00 AM

📌 一句话摘要

Databricks 与 SAP 合作推出 SAP Databricks，集成 SAP 数据与 Databricks 平台，提供人工智能、数据仓库和数据工程能力，或将革新企业数据分析。

📝 详细摘要

Databricks 宣布与 SAP 建立深度合作伙伴关系，推出 SAP Databricks，由 SAP 在 Azure、AWS 和 GCP 上作为 SAP 业务数据云的一部分销售。SAP Databricks 结合了 SAP 数据与 Databricks 平台，为客户提供由 Unity Catalog 管理的人工智能、数据仓库和数据工程能力。它简化了 SAP 数据与现代数据平台的使用，使企业能够通过 Delta Sharing 轻松地将 SAP 数据与其他数据结合，以提升团队生产力。Databricks 计划投资 2.5 亿美元来支持客户部署和迁移，从而释放 SAP 数据的业务价值。

💡 主要观点

  1. SAP Databricks 集成 SAP 数据与 Databricks 平台，提供全面的数据和人工智能能力。 这种合作关系解决了使用 SAP 数据进行现代数据平台分析的挑战，提供了从数据仓库到人工智能开发的完整解决方案。
  2. 企业可以通过 Delta Sharing 轻松集成 SAP 数据和其他数据。 SAP 业务数据云使客户能够在 SAP Databricks 和现有的 Databricks 部署中释放 SAP 数据的业务价值，以提升团队生产力。
  3. Databricks 计划大力投资于支持 SAP Databricks 的部署和迁移。 Databricks 将投资 2.5 亿美元，以帮助客户和集成商成功部署 SAP Databricks 并释放 SAP 数据的业务价值。
  4. SAP Databricks 通过对关键业务数据进行推理来实现数据智能。 该平台的独特价值在于它不仅能够提供通用智能，还能提供能够对关键业务数据进行推理的智能。

💬 文章金句

  • SAP Databricks 为客户提供人工智能、数据仓库和数据工程能力，所有这些都由 Unity Catalog 管理。
  • SAP 业务数据云使客户能够释放富含上下文的 SAP 数据的洞察力和业务价值。通过 Delta Sharing，这些价值可以在 SAP Databricks 以及所有现有和新的 Databricks 部署中实现。
