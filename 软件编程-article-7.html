本文介绍如何在 Windows 系统上搭建免费的本地 LaTeX 环境。

📝 详细摘要

本文详细介绍如何在 Windows 操作系统上搭建免费的本地 LaTeX 环境，并提供逐步安装和配置指南。文章首先指出，在线 LaTeX 平台如 Overleaf 在大型项目和协作方面存在局限性。本地环境结合 Git，则能有效解决这些问题。随后，文章讲解了所需工具的安装与配置，包括 VS Code、MiKTeX 和 Strawberry Perl。此外，还介绍了 VS Code 中 LaTeX Workshop 和 Rewrap 插件的使用。最后，文章说明如何使用 VS Code 编译 LaTeX 文件，并提供从 PDF 跳转回代码的技巧。通过本文，读者可以在本地免费运行 LaTeX 项目，并利用 Git 进行版本控制和协作。

💡 主要观点

  1. 本地 LaTeX 环境可解决在线平台在大型项目上的限制。 在线 LaTeX 平台免费版本在处理大型项目或多人协作时存在性能瓶颈。本地环境则不受此限制，并能与 Git 进行版本控制。
  2. MiKTeX 是一款适用于 Windows 的高性能免费 LaTeX 编译器。 MiKTeX 完全免费，支持所有主流操作系统，并自带 IDE，是配置本地 LaTeX 环境的关键组件。
  3. VS Code 插件 LaTeX Workshop 可提升 LaTeX 编辑效率。 LaTeX Workshop 提供代码高亮、智能补全、实时编译等功能，Rewrap 插件则能便捷地进行长行代码的自动换行与格式化，从而显著提升开发效率。
  4. Perl 是 Windows 环境下编译 LaTeX 文件的必要依赖。 由于 Windows 操作系统不自带 Perl 编译器，因此安装 Strawberry Perl 是在 Windows 上配置 LaTeX 环境的关键步骤。
