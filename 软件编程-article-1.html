Java 近期新闻：JDK 24 RC1、JDK Mission Control、Spring、Hibernate、Vert.x

3/2/2025, 2:26:28 AM

📌 一句话摘要

Java 生态系统近期更新汇总，包括 JDK、Spring、Hibernate、Vert.x 等框架的新版本发布及特性。

📝 详细摘要

本文汇总了近期 Java 生态系统的关键版本更新，涵盖 JDK 24/25、Spring 框架 / Data / Tools、Open Liberty、Micronaut、Hibernate、Eclipse Vert.x、Micrometer、Piranha Cloud、Project Reactor、JHipster 和 Gradle 等项目的更新，包括新特性、问题修复和依赖项升级。文章为 Java 开发者提供了一份全面的生态更新概览。

💡 主要观点

  1. JDK 24 发布候选版本，JDK 25 发布早期访问版本 JDK 24 预计将于 2025 年 3 月 GA 发布，包含 24 个最终确定的功能。JDK 25 的早期访问构建版本也已发布，修复了多个问题。
  2. Spring 生态系统迎来多个组件的版本更新 Spring Framework、Spring Data 和 Spring Tools 均发布了新版本，包含新功能、问题修复和依赖项升级，进一步提升开发效率。
  3. 多个 Java 框架和库发布新版本 Hibernate、Vert.x、Micronaut 等常用 Java 框架和库也发布了新版本，带来了性能改进、新特性和问题修复，值得开发者关注。

💬 文章金句

  • JDK 24 进入第一个发布候选阶段，因为 Build 36 中没有未解决的 P1 级错误。
  • Spring Data 2025.0.0 的第一个里程碑版本发布，包含新功能，例如：通过 MongoDB Atlas 和 Cassandra Vector Search 为 MongoDB 和 Cassandra 提供向量搜索支持
  • Hibernate ORM 7.0.0 的第四个 Beta 版本发布，包含以下功能：迁移到 Jakarta Persistence 3.2 规范（这是针对 Jakarta EE 11 的最新版本）
