马斯克发布 Grok 3，具备防“蒸馏”技术

2/19/2025, 3:02:21 PM

📌 一句话摘要

xAI 发布 Grok 3，提升推理能力并采用防蒸馏技术，挑战 GPT-4o。

📝 详细摘要

埃隆 · 马斯克的 xAI 公司发布了最新旗舰模型 Grok 3，该模型在性能上显著提升，尤其是在 AIME 和 GPQA 等基准测试中超越了 GPT-4o，并引入了独特的防“蒸馏”技术以保护知识产权。Grok 3 推出了 Grok 3 Reasoning 和 Grok 3 mini Reasoning 两种变体，增强了推理能力。为防止知识提取，Grok 3 应用程序隐藏了部分推理过程，并推出了 DeepSearch 功能。X 平台 Premium+ 订阅用户将首先获得 Grok 3 的访问权限，xAI 还计划在未来几个月内开源 Grok 2。马斯克表示，Grok 3 致力于追求绝对真理，并努力调整至更接近政治中立的立场。

💡 主要观点

  1. Grok 3 性能显著提升，尤其在推理能力上。 Grok 3 在 AIME 和 GPQA 等基准测试中超越 GPT-4o，并通过推理模型和“超级大脑”模式增强了问题解决能力。
  2. xAI 引入防“蒸馏”技术以保护知识产权。 为防止其他模型提取知识，Grok 3 应用程序隐藏了推理过程，并采取其他措施防止模型被“蒸馏”。
  3. Grok 3 将通过多种方式向用户开放。 X 平台的 Premium+ 订阅用户将首先获得访问权限，未来还将通过企业 API 提供，并增加语音模式。
  4. Grok 3 致力于追求绝对真理，并努力调整至更接近政治中立的立场。 马斯克承诺要将 Grok 调整得更接近政治中立，但具体实现方式和潜在后果尚不明确。

💬 文章金句

  • Grok 3 比 Grok 2 强大了一个数量级，这是一个追求绝对真理的人工智能，即使这种真理有时与政治正确相冲突。
  • 我们的总体方法是，当新版本完全推出后，我们会开源上一个版本。当 Grok 3 成熟且稳定时，可能在未来几个月内，我们就会开源 Grok 2。
