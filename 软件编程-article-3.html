NVIDIA GB200 GPU 赋能：全新 A4X 虚拟机亮相

2/20/2025, 12:00:00 AM

📌 一句话摘要

Google Cloud 推出采用 NVIDIA GB200 NVL72 GPU 的 A4X 虚拟机，为大规模 AI 模型推理与训练带来突破性进展。

📝 详细摘要

Google Cloud 推出 A4X 虚拟机，其核心是 NVIDIA GB200 NVL72。该系统配备 72 个 NVIDIA Blackwell GPU 和 36 个基于 Arm 的 NVIDIA Grace CPU，并通过第五代 NVIDIA NVLink 互联。A4X 虚拟机满足了推理模型对计算和内存的严苛需求，显著提升 AI 性能与精度。其主要特性包括：NVIDIA GB200 NVL72 加速器、NVIDIA Grace CPU、卓越的训练性能、出色的可扩展性，以及针对 AI 推理的深度优化。A4X 虚拟机与 Google Cloud 服务（如 Cloud Storage FUSE、Google Kubernetes Engine (GKE) 和 Vertex AI Platform）集成。借助 Google 的第三代液冷基础设施和 AI Hypercomputer 集群，A4X 确保了峰值计算性能和大规模 AI 工作负载的有效管理。NVIDIA DGX Cloud 也将在 A4X 虚拟机上提供。A4X 虚拟机专为应对最复杂的 AI 工作负载而生，A4 虚拟机则为各种 AI 模型架构提供卓越的性能和广泛的适用性。

💡 主要观点

  1. A4X 虚拟机与 Google Cloud 服务原生集成，简化了 AI 工作负载管理。 集成 Cloud Storage FUSE、GKE 和 Vertex AI 等服务，加速了 AI 项目的开发与部署。同时，优化资源利用率，充分发挥 Google Cloud 生态系统的优势。
  2. A4X 虚拟机采用 NVIDIA GB200 NVL72，可将 72 个 GPU 整合为统一的计算单元。 这种统一的架构有助于跨并发推理请求的多模态推理的低延迟响应，从而提高 AI 模型在处理复杂任务中的效率。
  3. A4X 虚拟机通过 NVLink 连接的 NVIDIA Grace CPU 优化训练性能。 Grace CPU 能够高效地执行检查点（Checkpointing）、卸载（Offloading）和重物化（Rematerializing）模型及优化器状态，这对于训练和部署大型模型至关重要。

💬 文章金句

  • 借助这一集成系统，A4X 虚拟机能够直接满足采用思维链的推理模型对海量计算和内存的需求，从而显著提升 AI 性能与精度。
  • A4X 虚拟机支持采用最新的分片（Sharding）和流水线（Pipelining）技术，在数万个 Blackwell GPU 上部署模型，从而最大限度地提高 GPU 利用率。
  • Google Cloud 提供由 NVIDIA B200 GPU 驱动的 A4 虚拟机，以及现在由 NVIDIA GB200 NVL72 驱动的 A4X 虚拟机。
