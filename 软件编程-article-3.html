MongoDB 的开源替代方案 FerretDB 发布 2.0 版本

2/22/2025, 6:40:28 AM

📌 一句话摘要

FerretDB 2.0 发布，作为 MongoDB 的开源替代方案，通过与 DocumentDB 集成，在性能、兼容性和功能上实现了显著提升。

📝 详细摘要

FerretDB 2.0 作为 MongoDB 的开源替代方案正式发布，该版本由 DocumentDB 驱动，在性能和兼容性方面实现了显著提升。通过与微软开源的 DocumentDB PostgreSQL 扩展集成，FerretDB 2.0 能够更好地支持 PostgreSQL 数据库，提升了在 Postgres 上运行文档数据库工作负载时的性能，并对现有应用程序提供更好的支持。FerretDB 2.0 基于 Apache 2.0 许可发布，与 MongoDB 的驱动程序和工具兼容，旨在为那些希望避开 SSPL 许可的开源项目和早期商业项目提供直接替代方案。此外，FerretDB 2.0 还带来了向量搜索能力以及复制支持，FerretDB Cloud 正在开发中，将提供托管数据库即服务选项。

💡 主要观点

  1. FerretDB 2.0 通过集成 DocumentDB 显著提升性能 利用微软的 DocumentDB PostgreSQL 扩展，FerretDB 2.0 在处理某些工作负载时速度提升高达 20 倍，性能与市场领先竞品相当。
  2. FerretDB 致力于成为 MongoDB 的开源替代方案 基于 Apache 2.0 许可，FerretDB 旨在为避开 SSPL 许可的项目提供直接替代方案，并保持与 MongoDB 驱动程序和工具的兼容性。
  3. FerretDB Cloud 将提供托管数据库即服务 FerretDB Cloud 正在开发中，最初将在 AWS 和 GCP 上提供，后续计划支持微软 Azure，为用户提供更便捷的部署和管理选项。

💬 文章金句

  • FerretDB 2.0 在性能和兼容性方面实现了巨大飞跃。
  • 随着微软开源 DocumentDB，我们比以往任何时候都更接近，能真正在行业内开展协作共同创建文档数据库的开放标准。
  • 寻找即用型 NoSQL 数据库的用户可以借助 FerretDB 中的现有解决方案......用户可以通过 Postgres 与 DocumentDB 进行交互，而 FerretDB 2.0 则提供了符合文档数据库协议的接口。
