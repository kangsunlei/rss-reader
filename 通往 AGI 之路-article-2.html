  • 《老外6万美金买的域名 Cursor.com 一年后卖了39.9万美元：聊聊 Cursor 早期的故事》老外 Mike Carson 于 2023 年 3 月花 6 万美金买下 Cursor.com，一年后以 39.9 万美元卖给 AI 编程工具 Cursor。
  • 《[Agentic AI] 搏一搏，$20变$500：一小时魔改Cursor变身Devin》
    • Devin 和 Agent Cursor 的差异：Devin 更像有章法的人，能全局规划、灵活使用工具、动态成长，适合复杂项目；Cursor 更像工具，稳定性和完成任务能力强，适合轻量级任务。
    • Agentic AI 的核心特质与技术架构：核心特质是使用工具和多步决策；技术架构包括调用工具的 LLM、明确成功标准、工具描述与标准化、Orchestrator 等方面。
    • 魔改 Cursor 接近 Devin：通过修改 .cursorrules 文件实现流程规划和自我进化，通过准备程序和控制 prompt 实现更多工具使用，Windsurf 在某些方面与 Cursor 不同。
    • 使用 Cursor 作为通用入口：可用于问答、接入私有文档 RAG、搜索、个性化等，具有方便沉淀文档等优势，但存在无 API 和移动端的局限。
