最低调的「AI 六小龙」阶跃星辰，在 DeepSeek 浪潮中交出了一份新答卷

2/21/2025, 8:51:47 PM

📌 一句话摘要

阶跃星辰 All in 多模态 Agent，与多家公司合作落地智能终端应用，并开源相关模型，探索 AGI 路径，其多模态能力是亮点。

📝 详细摘要

文章介绍了 AI 六小龙之一的阶跃星辰在 AGI 探索上的最新进展。阶跃星辰 CEO 姜大昕认为智能终端是 Agent 技术大规模应用的关键载体，并着重发展多模态和推理能力，尤其在多模态 AI 方面处于领先地位。阶跃星辰开源了 Step-Video-T2V 视频生成模型和 Step-Audio 语音模型，以及与清华大学共同研发的开源推理模型 Open-Reasoner-Zero，积极拥抱开源。同时，阶跃星辰与吉利（智能座舱“可见即可说”）、OPPO、TCL、智元机器人等公司合作，将多模态 Agent 应用于汽车、手机、IoT 和具身智能等场景。文章还提到了阶跃星辰对 AGI 发展阶段的设想，即模拟世界、探索世界和归纳世界。

💡 主要观点

  1. 阶跃星辰 All in 多模态 Agent 阶跃星辰将多模态 Agent 作为其战略选择，All in 多模态 Agent， 探索 AGI 新路径。
  2. 智能终端是 Agent 技术大规模应用的关键载体 阶跃星辰将 AI 落地的重点押注在智能终端，通过多模态能力和推理能力结合，实现更智能的人机交互。
  3. 多模态能力和推理能力是智能体的两大关键要素 多模态让智能体充分地感知和理解世界，推理能力使智能体能够进行长思维链的慢思考，主动规划、尝试、反思。
  4. 阶跃星辰通过合作将多模态 Agent 落地到多个场景 与吉利合作开发智能座舱 Agent，实现“可见即可说”；与 OPPO 共建“一键问屏”功能；与 TCL 等厂商合作实现 IoT 设备间联动。
  5. 阶跃星辰积极拥抱开源 开源 Step-Video-T2V 视频生成模型和 Step-Audio 语音模型，以及与清华大学共同研发的开源推理模型 Open-Reasoner-Zero。

💬 文章金句

  • 智能终端将成为 Agent （智能体）技术大规模应用的关键载体。
  • 们朝着实现 AGI 的目标稳步前进，目前已进入 Agent（智能体）的发展阶段。
  • 智能终端正在迈入 Agent 时代。和过去最大的不同在于：本地化 AI 推理能力使终端从被动响应指令的工具，升级为主动理解需求、跨应用协同的「智能伙伴」。
