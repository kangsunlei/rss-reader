利用 BigQuery ML 运行开源 LLM：实践指南

2/19/2025, 12:00:00 AM

📌 一句话摘要

本文演示如何通过 BigQuery ML 和 Vertex AI Model Garden 集成开源 LLM，简化数据分析并丰富模型选择。

📝 详细摘要

本文介绍了如何将 Vertex AI Model Garden 中的开源 LLM 集成到 BigQuery ML 中，从而简化数据分析任务并扩展模型选择。它详细介绍了在 BigQuery ML 中使用 Meta Llama 3.3 70B 模型进行部署、推理和分析数据的步骤。该集成简化了在 BigQuery 中利用 LLM 进行数据分析的过程，从而可能降低使用门槛并提高效率。通过使用 SQL 和 LLM，可以从非结构化数据（如医疗记录）中提取年龄、性别和疾病等结构化信息。

💡 主要观点

  1. BigQuery ML 现在支持来自 Vertex AI Model Garden 的开源 LLM。 开发人员可以使用来自 Hugging Face 等平台的开源 LLM，从而扩展模型选择并利用 BigQuery ML 进行数据分析。
  2. 在 BigQuery ML 中使用开源 LLM 涉及三个主要步骤。 这些步骤包括在 Vertex AI 上部署模型，在 BigQuery 中创建远程模型，以及使用 SQL 语句执行推理和分析。
  3. SQL 语句和 LLM 可以从非结构化数据（如医疗记录）中提取结构化信息。 该示例演示了从医疗记录数据集中提取患者的年龄、性别和疾病信息，以进行进一步分析。

💬 文章金句

  • BigQuery Machine Learning 允许您使用大型语言模型 (LLM)，例如 Gemini，使用熟悉的 SQL 语法对您的数据执行实体提取、情感分析、翻译、文本生成等任务。
  • 今天，我们将通过支持来自 Vertex AI Model Garden 的任何开源 LLM（包括从 Hugging Face 部署或经过调整的 OSS 模型）来扩展此功能。
  • 您现在可以针对 BigQuery ML 中的此模型执行推理。
