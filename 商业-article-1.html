DeepSeek R1 API 实测，哪家服务商最靠谱？

2/13/2025, 12:12:28 PM

📌 一句话摘要

评测 DeepSeek R1 API 服务商，推荐优质供应商并提供自测工具。

📝 详细摘要

文章评测了国内主流云平台提供的 DeepSeek R1 API 服务，旨在帮助开发者选择靠谱的服务商。文章给出选择 R1 服务商的关键指标：原版模型（6000 亿参数）、64k+ 的 Context Window 以及 10w 以上的 TPM。通过小输入 (100 token) 和大输入 (14000 token) 两种场景，对比了各平台 R1 和 V3 模型的 TPS（每秒 Token 生成速度）和 TTFT（首 token 出现等待时间）指标。评测结果显示，火山方舟和硅基流动在小输入测试中表现出色，硅基流动 R1 的首 token 等待时间为 0.2 秒，火山方舟在大输入测试中 R1 的 TPS 达到 24.90t/s。最后，文章还提供了本地部署的测试工具，方便开发者自行验证。

💡 主要观点

  1. 选择 R1 服务商需关注模型版本、窗口大小和 TPM 限制 非原版模型、过小的 Context Window 和 TPM 限制会严重影响 R1 的使用体验和效果，需要仔细甄别。
  2. 火山方舟和硅基流动在 R1/V3 服务上表现领先 两家平台在速度指标上表现优异，技术实力和服务质量值得信赖，是国内 R1/V3 服务的首选。
  3. 文章提供了使用 auto-coder 工具本地测试 R1/V3 模型的详细步骤和命令 开发者可以使用 auto-coder 工具，根据文章提供的命令和配置方法，自行测试不同平台 R1/V3 模型的性能。

💬 文章金句

  • R1/V3 都只有一个规格的版本，都是 6000+亿参数的，不存在所谓满血不满血。
  • 作为用户，你区分一个供应商是不是真的良心供应商，先看模型不是是原版的（6000 亿参数规模），其次再看窗口大小（64k+ 是正常），最后看 TPM(一般要 10w 以上)。
  • 国内 R1/V3 提供上，表现最好的是就是火山方舟和硅基流动，基本也和我之前的认知相同，他们两家技术实力强悍，行动也都很早，而且效果上也和官方有在对齐，基本值得信赖。
