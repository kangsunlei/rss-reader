体验支持远程开发功能的 Toolbox App 2.6 EAP | The Toolbox App Blog

📌 一句话摘要

JetBrains Toolbox App 2.6 EAP 引入远程开发支持，使用户可以通过 OpenSSH 安全地管理在 Linux、Windows 和 macOS 主机上的 JetBrains 工具和项目。

📝 详细摘要

JetBrains Toolbox App 2.6 EAP 最新更新引入远程开发功能。此 Early Access Program (EAP) 允许用户不仅可以在本地，还可以在运行 Linux、Windows 或 macOS 的远程服务器上管理 JetBrains IDE 和项目。该应用通过 OpenSSH 提供安全且可定制的 SSH 连接，与现有 SSH 配置无缝集成，支持 ProxyJump、多因素身份验证和自定义 IdentityFile 配置。用户可以轻松在远程主机上安装、访问、更新和监控 JetBrains IDE，保持一致的工作流程。Toolbox App 还支持导入现有 SSH 配置，并通过插件 API 扩展以支持编排解决方案。

💡 主要观点

  1. Toolbox App 现在支持在 Linux、Windows 和 macOS 上的远程开发。 此更新允许在 Linux、Windows 和 macOS 主机上管理 JetBrains 工具和项目，从而实现一致的开发体验。
  2. OpenSSH 集成确保安全且可定制的远程连接。 利用标准 OpenSSH 可执行文件提供企业级安全性和灵活性，支持各种 SSH 配置。
  3. 插件 API 扩展了 Toolbox App 的功能，以用于编排解决方案。 可用的插件 API 允许扩展 Toolbox App 以适应特定的编排需求，并且正在开发用于流行编排器的插件。

💬 文章金句

  • 最新更新后，Toolbox App 支持远程开发，让您可以在本地和远程服务器上管理 JetBrains 工具和项目。
  • Toolbox App 的 SSH 支持由 OpenSSH 提供，为远程开发带来企业级安全性和灵活性。
  • Toolbox App 使用您熟悉的 OpenSSH 工具，使远程连接设置更简单、安全，并能完全自定义以适应您的工作流程。
