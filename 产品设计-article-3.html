第一批 DeepSeek 开发者，已经开始逃离了

2/20/2025, 4:02:02 PM

📌 一句话摘要

DeepSeek 以高性价比吸引开发者，但因服务不稳定导致早期开发者流失，并面临大厂竞争压力。

📝 详细摘要

国产大模型 DeepSeek 曾因推理能力和高性价比迅速获得市场认可，但随着用户激增，服务器过载问题导致服务不稳定，早期开发者面临 API 调用频繁失败（有时持续数天）等问题，缺乏提前通知和售后服务，因此转向 ChatGPT 等更稳定的平台。DeepSeek 虽然在中文理解和写作想象力方面有优势，但其稳定性和价格优势正在减弱，面临百度、腾讯、 OpenAI 等大厂在技术、价格和服务上的竞争，大厂通过价格战和服务升级等措施争夺市场。DeepSeek 在多模态支持等方面存在技术短板，客户服务体系也有待完善，加强基础设施建设是潜在的解决方案。

💡 主要观点

  1. DeepSeek 因服务不稳定导致早期开发者流失 服务器过载、API 调用失败、缺乏提前通知和售后服务等问题，迫使开发者转向其他更稳定的平台。
  2. DeepSeek 面临大厂在技术、价格和服务上的竞争 百度、腾讯、 OpenAI 等都在追赶 DeepSeek 的深度推理能力，且大厂在价格和服务上更具优势，例如通过云服务绑定客户。
  3. DeepSeek 的性价比优势正在减弱 大厂 API 价格持续走低，DeepSeek 在发票开具、通用能力支持等方面存在不足，性价比标签面临挑战。
  4. DeepSeek 在技术上存在短板，尤其是在多模态支持方面 与 ChatGPT 和 Gemini 2.0 相比，DeepSeek 在支持视频、语音、文档、图片等多种输入模态方面仍有差距。
  5. DeepSeek 的客户服务体系有待完善 缺乏专门的客户经理对接和便捷的发票开具流程，影响了开发者体验。

💬 文章金句

  • ChatGPT 虽然贵一点，但至少稳定。
  • 现在已经有几十个平台有满血版 DeepSeek R1 了。
  • 每 12 个月，推理成本就可以降低 90%以上。
  • 就像家门口有一家小店，你是老顾客，办了卡，一直和老板相处很好。突然有一天，小店被评为米其林餐厅，老板把老顾客抛在一边，之前办的卡也不认了。
